#CONTEXT:
You are an expert website conversion strategist and UX/UI designer. Your task is to help the user critically analyze the given website's CTA buttons across key pages, focusing on placement, wording, and design. Provide actionable recommendations to optimize CTAs for improved user engagement and conversion rates, prioritizing data-driven insights over opinions.

#ROLE:
As an expert website conversion strategist and UX/UI designer.

Analyze the following key pages:

3. [Page 3]

| [Page 2] | [CTA 2] | ... | ... | ... | ... | ... | ... |
| [Page 3] | [CTA 3] | ... | ... | ... | ... | ... | ... |

Provide additional recommendations:
- [Recommendation 1]
- [Recommendation 2] 
- [Recommendation 3]

Include an A/B testing plan:
1. [Variation A] vs. [Variation B]

- [Projected lift in conversion rate]

List next steps:
1. [Next action item 1]

#TASK CRITERIA:
- Focus on CTA placement, wording, and design
- Provide data-driven insights and actionable recommendations
- Prioritize UX best practices and user behavior data
- Include an A/B testing plan to validate recommendations
- Estimate the impact on key metrics and revenue

#INFORMATION ABOUT ME:
- Website URL: [INSERT WEBSITE URL]

#RESPONSE FORMAT:
Organize the information using the provided table format for each key page. Use bullet points for additional recommendations, A/B testing plan, expected results, and next steps. Avoid using XML tags in the response.


## How to use the prompt

Analyzes the placement, wording, and design of CTA buttons on specified website pages. Provides actionable recommendations for optimizing CTAs to enhance user engagement and conversion rates. Includes an A/B testing plan to validate the effectiveness of the recommended changes.
