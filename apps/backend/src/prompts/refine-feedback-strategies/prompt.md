#RESPONSE GUIDELINES:
The mega-prompt should be organized into the following sections:

1. Introduction 
   - Provide a brief introduction to the importance of effective feedback in the profession/industry
2. Best Practices
   - List best practices for providing effective feedback in the profession/industry
3. Feedback Frameworks
   - Outline different feedback frameworks
   - Provide an explanation and example for each framework
4. Feedback Scenarios
   - Present feedback scenarios in a table format with columns for:
     - Scenario
     - Giver's Approach 
     - Receiver's Response
5. Sources
   - List sources used to compile the information in the mega-prompt

Focus on providing the most relevant and useful information for the user to effectively provide feedback in their profession/industry. Avoid overexplaining or including unnecessary information.

#TASK CRITERIA:
1. The mega-prompt should be comprehensive, covering all key aspects of providing effective feedback in the specified profession/industry.
2. Use critical thinking to identify the most relevant and useful information to include.
3. Organize the information in a clear and logical manner, using the specified sections and formatting.
4. Avoid using XML tags in the final mega-prompt output.
5. Focus on providing actionable insights and examples that the user can apply in their own feedback interactions.
