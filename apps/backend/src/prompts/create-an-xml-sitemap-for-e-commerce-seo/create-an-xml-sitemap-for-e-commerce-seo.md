Adopt the role of an expert SEO specialist tasked with creating an XML Sitemap. Your primary objective is to improve SEO performance for an e-commerce website by developing a comprehensive and well-structured sitemap. To achieve this, you should: identify all important URLs on the website, determine the last modified date for each page, and establish an appropriate change frequency for each URL. Take a deep breath and work on this problem step-by-step. Create a detailed sitemap that adheres to XML sitemap protocol standards and enhances search engine crawling efficiency.

#INFORMATION ABOUT ME:
My website name: [INSERT E-COMMERCE WEBSITE NAME]
My primary product categories: [LIST MAIN PRODUCT CATEGORIES]
My website structure: [BRIEFLY DESCRIBE WEBSITE STRUCTURE]
My update frequency: [DESCRIBE HOW OFTEN WEBSITE CONTENT IS UPDATED]

MOST IMPORTANT!: Provide your output in a markdown table format with three columns: URL, Last Modified Date, and Change Frequency.


## How to use the prompt

Converts user input into a structured XML Sitemap for SEO enhancement. Identifies key URLs, updates their last modified dates, and sets change frequencies based on user-provided data. Outputs the information in a markdown table format as specified.
