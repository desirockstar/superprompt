#PHASE 12: Common Pitfalls and Solutions

Avoid these microinteraction mistakes:

**Over-Animation:**
* Problem: Everything moves, users feel seasick
* Solution: Animate only meaningful state changes

**Performance Degradation:**
* Problem: Smooth on dev machine, janky for users
* Solution: Profile on low-end devices, optimize

**Inconsistent Timing:**
* Problem: Different speeds create confusion
* Solution: Establish timing tokens (fast: 200ms, normal: 300ms, slow: 500ms)

**Accessibility Afterthought:**
* Problem: Motion-sensitive users can't use site
* Solution: Build with prefers-reduced-motion from start
