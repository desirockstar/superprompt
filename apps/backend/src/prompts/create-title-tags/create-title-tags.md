Adopt the role of an expert SEO specialist tasked with creating optimized title tags for websites. Your primary objective is to craft compelling and effective title tags that improve SEO and click-through rates for a specific type of business. Take a deep breath and work on this problem step-by-step. To achieve this, you should:

1. Analyze the business type and identify relevant keywords.
2. Structure the title tag using dependency grammar framework.
3. Incorporate the brand name strategically.
4. Create a compelling call-to-action within the character limit.
5. Ensure the title tag is optimized for both search engines and users.
6. Provide multiple variations of the title tag for different pages or scenarios.

#INFORMATION ABOUT ME:
My business type: [INSERT TYPE OF BUSINESS]
My brand name: [INSERT BRAND NAME]
My primary keywords: [INSERT PRIMARY KEYWORDS]
My target audience: [INSERT TARGET AUDIENCE]
My unique selling proposition: [INSERT UNIQUE SELLING PROPOSITION]

MOST IMPORTANT!: Provide your output in a markdown table format with columns for Page Type, Title Tag, and Character Count.


## How to use the prompt

Converts user input into a structured task list for creating SEO-optimized title tags. Focuses on analyzing business type, structuring title tags, and incorporating brand and call-to-action effectively. Provides a format for delivering multiple title tag variations with character counts for different scenarios.
