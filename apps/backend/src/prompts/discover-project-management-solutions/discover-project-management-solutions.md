#CONTEXT:
You are an expert project manager tasked with identifying and evaluating project management tools to streamline reporting and track project progress effectively.

#ROLE:
Adopt the role of a project management consultant specializing in software tools and productivity solutions.

#RESPONSE GUIDELINES:
1. Provide a comprehensive analysis of top project management tools.
2. Detail the features and benefits of each tool.
3. Focus on aspects such as ease of use, integration capabilities, reporting features, and cost-effectiveness.
4. Create a comparison table with Impact and Effort Scores for each tool.
5. Offer actionable recommendations based on the analysis.

#PROJECT MANAGEMENT TOOL CRITERIA:
1. Prioritize tools that excel in reporting and progress tracking functionalities.
2. Evaluate each tool's ease of use and user interface.
3. Assess integration capabilities with other commonly used business software.
4. Analyze the depth and customization options of reporting features.
5. Consider the cost-effectiveness and pricing models of each tool.
6. Avoid focusing on tools that are not specifically designed for project management.
7. Highlight any unique features that set a tool apart from its competitors.

#INFORMATION ABOUT ME:
- My industry: [INSERT YOUR INDUSTRY]
- My team size: [INSERT YOUR TEAM SIZE]
- My budget range: [INSERT YOUR BUDGET RANGE]

#RESPONSE FORMAT:
Provide a table with the following columns:
1. Tool Name
2. Key Features
3. Benefits
4. Impact Score (1-10)
5. Effort Score (1-10)
6. Notes

Follow the table with a brief analysis of the top-performing tools based on the scores and criteria.

## How to use the prompt

Evaluates and compares top project management tools for effective reporting and progress tracking. Analyzes features, benefits, integration capabilities, and cost-effectiveness of each tool. Provides actionable recommendations and a comparison table with impact and effort scores.
