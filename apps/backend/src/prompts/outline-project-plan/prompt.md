Adopt the role of an expert project manager tasked with developing a comprehensive project plan for a grant proposal. Your primary objective is to create a detailed outline of key milestones, deliverables, and resources needed in a structured table format. Take a deep breath and work on this problem step-by-step. Begin by analyzing the grant proposal type and identifying critical components. Then, break down the project into logical phases, determining specific milestones and associated deliverables for each phase. Consider the resources required for each milestone, including personnel, equipment, and funding. Establish realistic timelines for each milestone, taking into account potential dependencies and constraints. Ensure that your project plan is comprehensive, achievable, and aligned with the grant proposal's objectives.

#INFORMATION ABOUT ME:
My grant proposal type: [INSERT GRANT PROPOSAL TYPE]
My project timeline: [INSERT PROJECT TIMELINE]
My available resources: [INSERT AVAILABLE RESOURCES]
My key stakeholders: [INSERT KEY STAKEHOLDERS]
My project goals: [INSERT PROJECT GOALS]

MOST IMPORTANT!: Present your output in a markdown table format with 4 columns: Milestone, Deliverable, Resource, and Timeline.


## How to use the prompt

Converts user input into a structured project plan for a grant proposal. Identifies and outlines key milestones, deliverables, and required resources in a table format. Establishes timelines and dependencies for each project phase aligned with the proposal's objectives.
