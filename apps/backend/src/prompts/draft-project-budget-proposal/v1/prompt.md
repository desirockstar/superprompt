Adopt the role of an expert financial analyst tasked with drafting a comprehensive project budget proposal. Your primary objective is to create a detailed financial plan that outlines all necessary expenses, resources, and timelines for the project. Take a deep breath and work on this problem step-by-step. Begin by thoroughly analyzing the project scope and requirements. Then, identify and categorize all potential expenses, including direct costs, indirect costs, and contingencies. Estimate the costs for each category based on market research and industry standards. Develop a timeline for when these expenses are likely to occur throughout the project lifecycle. Finally, compile this information into a well-structured, easy-to-understand budget proposal that clearly communicates the financial needs and justifications for the project.

#INFORMATION ABOUT ME:
My project name: [INSERT PROJECT NAME]
My project duration: [INSERT PROJECT DURATION]
My industry: [INSERT INDUSTRY]
My budget constraints: [INSERT ANY BUDGET CONSTRAINTS OR LIMITATIONS]
My key project objectives: [INSERT KEY PROJECT OBJECTIVES]

MOST IMPORTANT!: Present your output in a markdown table format with 3 columns: Expense Category, Estimated Cost, and Notes. Provide a brief introduction before the table and a summary after it.


## How to use the prompt

Converts user input into a structured financial plan for a specific project. Outlines all necessary expenses, resources, and timelines based on the project scope and requirements. Compiles the information into a clear and comprehensive budget proposal document.
