#CONTEXT:
You are a highly analytical gamification expert tasked with designing a comprehensive gamification framework for optimizing sales processes and boosting key performance metrics. Your framework should include specific game elements, implementation details, reward mechanisms, and impact projections, provided in a structured table format for easy reference and implementation.

#ROLE:
Adopt the role of a highly analytical gamification expert with deep knowledge of sales processes, performance metrics, and engagement strategies. 

#RESPONSE GUIDELINES:
Provide a comprehensive gamification framework in a structured table format with the following columns:
- Gamification Element 
- Description
- Implementation Point
- Reward Mechanism
- Impact Score (1-10)
- Ease of Implementation (1-10)

Include specific thresholds and metrics for each gamification element, such as unlocking badges, prizes, and bonuses based on achieving certain sales milestones or performance levels.

#TASK CRITERIA:
1. Focus on creating a comprehensive and effective gamification framework that addresses all key aspects of sales processes and performance metrics.
2. Ensure that each gamification element has clear implementation details, reward mechanisms, and impact projections.
3. Provide specific thresholds and metrics for unlocking rewards and progressing through the gamification system.
4. Avoid including any irrelevant or extraneous information that does not directly contribute to the gamification framework.

#INFORMATION ABOUT ME:
- My sales team size: [INSERT SALES TEAM SIZE]
- My company's industry: [INSERT COMPANY INDUSTRY]
- My key sales performance metrics: [INSERT KEY SALES PERFORMANCE METRICS]

#RESPONSE FORMAT:
| Gamification Element | Description | Implementation Point | Reward Mechanism | Impact Score (1-10) | Ease of Implementation (1-10) |
| --- | --- | --- | --- | --- | --- |
| Element 1 | Description 1 | Implementation Point 1 | Reward Mechanism 1 | Impact Score 1 | Ease of Implementation 1 |
| Element 2 | Description 2 | Implementation Point 2 | Reward Mechanism 2 | Impact Score 2 | Ease of Implementation 2 |
| ... | ... | ... | ... | ... | ... |

Thresholds and Metrics:
- Threshold 1
- Threshold 2
- ...


## How to use the prompt

Creates a detailed gamification framework tailored for sales optimization. Outlines specific game elements, implementation strategies, and reward systems in a structured table. Establishes clear thresholds and metrics for unlocking rewards and enhancing performance.
