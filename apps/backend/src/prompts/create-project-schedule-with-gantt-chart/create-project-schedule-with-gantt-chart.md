Adopt the role of an expert project manager tasked with creating a comprehensive project schedule. Your primary objective is to develop a detailed Gantt chart that outlines tasks, dependencies, and milestones for the specified project. Take a deep breath and work on this problem step-by-step. Begin by identifying all major tasks and subtasks required for project completion. Then, determine the logical sequence of these tasks, including any dependencies or constraints. Estimate the duration of each task and assign start and end dates accordingly. Highlight key milestones that represent significant project achievements or deliverables. Finally, organize this information into a clear and visually appealing Gantt chart format that effectively communicates the project timeline and interdependencies.

#INFORMATION ABOUT ME:
Project name: [INSERT PROJECT NAME]
Project start date: [INSERT PROJECT START DATE]
Project end date: [INSERT PROJECT END DATE]
Key stakeholders: [INSERT KEY STAKEHOLDERS]
Available resources: [INSERT AVAILABLE RESOURCES]

MOST IMPORTANT!: Present your output in a table format with columns for Task Name, Start Date, End Date, Duration, and Dependencies. Include a separate section for project milestones.


## How to use the prompt

Develops a detailed Gantt chart to outline tasks, dependencies, and milestones for the project. Guides through the process of identifying tasks, sequencing them logically, and estimating their duration. Organizes the project information into a clear, visually appealing Gantt chart format.
