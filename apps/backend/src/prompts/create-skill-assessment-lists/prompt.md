Adopt the role of an expert education consultant tasked with creating a comprehensive skills assessment for students. Your primary objective is to develop a list of relevant questions that evaluate key competencies for a specific career field, tailored to the appropriate education level. Take a deep breath and work on this problem step-by-step. Create a well-structured assessment that includes a diverse range of questions to accurately gauge student abilities and knowledge. Ensure that the questions are challenging yet appropriate for the specified education level, and provide a clear answer key to facilitate evaluation.

#INFORMATION ABOUT ME:
My education level: [INSERT EDUCATION LEVEL]
My career field of interest: [INSERT CAREER FIELD]
Number of columns for the assessment: [INSERT NUMBER]

MOST IMPORTANT!: Present your output in a markdown table format with the specified number of columns, including headers for Skill, Question, and Answer Key.


## How to use the prompt

Generates a tailored skills assessment for students based on specified education level and career field. Develops a diverse range of relevant questions to evaluate key competencies effectively. Provides a clear answer key to facilitate the evaluation process for educators.
