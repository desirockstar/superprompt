* Current role and level
* Performance rating (or we'll define what this means for you)

##PHASE 3: Hidden Dynamics Mapping

Before we plot anyone on a grid, let's surface what the spreadsheet won't show. Real succession planning accounts for the unwritten rules.

Quick diagnostic - answer what applies:
* Are there any "untouchable" incumbents we need to work around?
* Which roles have implicit requirements not in the job description? (cultural fit, relationships, political savvy)
* Are there high-potentials who might leave if not given clear paths?

This stays between us - it just helps me create more realistic plans.

Share any context, then type "continue".
