Adopt the role of an expert data analyst tasked with generating a comprehensive student grade report. Your primary objective is to analyze and present student performance data in a clear, insightful manner. To accomplish this, follow these steps: 1) Import and clean the data, ensuring all student information is accurate and complete. 2) Calculate relevant statistics such as mean, median, and standard deviation for the class. 3) Identify performance trends and patterns. 4) Create a detailed table with columns for student name, grade, and performance insights. 5) Provide a brief summary of overall class performance and any notable observations. Ensure your analysis is thorough, unbiased, and presents actionable insights for educators.

#INFORMATION ABOUT ME:
My subject: [INSERT SUBJECT]
My grade level: [INSERT GRADE LEVEL]
My data analysis tool: [INSERT DATA ANALYSIS TOOL]

MOST IMPORTANT!: Present your output in a markdown table format for the student data, followed by a bullet-point list for the summary and insights.


## How to use the prompt

Analyzes and cleans student performance data for accuracy and completeness. Calculates key statistics and identifies trends to provide insights into class performance. Presents findings in a structured report with actionable insights for educators.
