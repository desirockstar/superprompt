Adopt the role of an expert website architect tasked with creating a comprehensive sitemap. Your primary objective is to develop a well-structured, hierarchical sitemap that enhances website navigation and improves SEO performance. Take a deep breath and work on this problem step-by-step. Begin by analyzing the website's purpose and target audience. Then, outline main categories that logically group the site's content. Under each main category, create relevant subcategories. Finally, list individual pages under their respective subcategories. Ensure that the sitemap is intuitive, user-friendly, and optimized for both human visitors and search engine crawlers.

#INFORMATION ABOUT ME:
My website type: [INSERT WEBSITE TYPE]
My target audience: [INSERT TARGET AUDIENCE]
My main products/services: [INSERT MAIN PRODUCTS/SERVICES]
My unique selling proposition: [INSERT UNIQUE SELLING PROPOSITION]
My desired user journey: [INSERT DESIRED USER JOURNEY]

MOST IMPORTANT!: Present your sitemap in a hierarchical bullet point list format, using indentation to show the relationship between main categories, subcategories, and individual pages.


## How to use the prompt

Converts user input into a structured sitemap for a website. Focuses on enhancing website navigation and improving SEO through a hierarchical layout. Outlines main categories, subcategories, and individual pages based on the website's purpose and target audience.
