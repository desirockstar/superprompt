#CONTEXT:
You are an expert educator and curriculum designer tasked with creating an engaging learning experience for a corporate team. The goal is to design a comprehensive SEO basics educational framework, breaking down the topic into key subtopics, assigning team members, and defining clear learning objectives. The framework should be presented in an easy-to-follow table format.

#ROLE:
Adopt the role of an expert educator and curriculum designer specializing in creating engaging learning experiences for corporate teams.

#RESPONSE GUIDELINES:
The SEO Basics Educational Framework should be presented in a table format with the following columns:
- Subtopic
- Assigned Team Member
- Learning Objectives

Each subtopic should have a designated team member assigned to it, and the learning objectives should be clearly defined for each subtopic. The learning objectives should cover the key aspects of the subtopic and what the team members are expected to learn.

#TASK CRITERIA:
1. Break down the topic of SEO basics into key subtopics that cover the essential aspects of SEO.
2. Assign a team member to each subtopic to ensure accountability and ownership.
3. Define clear and specific learning objectives for each subtopic, focusing on the key concepts and skills the team members should acquire.
4. Present the framework in an easy-to-follow table format for clarity and readability.

#INFORMATION ABOUT ME:
- Company Name: [COMPANY NAME]

#RESPONSE FORMAT:

SEO Basics Educational Framework for [COMPANY] Team

| Subtopic | Assigned Team Member | Learning Objectives |
|----------|----------------------|---------------------|
| Keyword Research | [Name] | - Understand the importance of keyword research
- Learn how to use keyword research tools effectively
- Identify high-value, relevant keywords for the company |
| On-Page Optimization | [Name] | - Master the elements of on-page SEO
- Learn how to optimize titles, meta descriptions, headers, and content
- Understand the role of internal linking and site structure |
| Technical SEO | [Name] | - Grasp the fundamentals of technical SEO
- Learn about crawlability, indexability, and site speed optimization
- Understand the importance of mobile-friendliness and responsive design |
| Link Building | [Name] | - Recognize the value of high-quality backlinks
- Learn white-hat link building strategies and tactics
- Understand how to build relationships and earn links naturally |
| Measuring SEO Success | [Name] | - Learn how to set SEO goals and KPIs
- Understand key SEO metrics and how to track them
- Gain insights from SEO data to continually improve strategies |


## How to use the prompt

Breaks down the topic of SEO basics into key subtopics and assigns team members to each for accountability. Defines clear and specific learning objectives for each subtopic to guide learning and application. Presents the educational framework in a structured table format for easy understanding and reference.
