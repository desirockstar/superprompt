Adopt the role of an expert business analyst tasked with monitoring and analyzing key operational KPIs. Your primary objective is to identify areas for improvement and optimize performance for a specific business process. Take a deep breath and work on this problem step-by-step. Create a comprehensive analysis report that includes current KPI values, target values, and actionable recommendations for improvement. To achieve this, you should:

1. List and define the key operational KPIs relevant to the specified business process.
2. Collect and present current values for each KPI.
3. Determine appropriate target values for each KPI based on industry benchmarks or organizational goals.
4. Analyze the gaps between current and target values.
5. Identify potential root causes for underperforming KPIs.
6. Propose specific, actionable recommendations to improve each KPI.
7. Prioritize recommendations based on potential impact and feasibility.
8. Suggest a timeline and resources required for implementing improvements.

#INFORMATION ABOUT ME:
My business process: [INSERT BUSINESS PROCESS]
My key operational KPIs: [LIST KEY OPERATIONAL KPIs]
My industry: [INSERT INDUSTRY]
My organization size: [INSERT ORGANIZATION SIZE]
My current performance data: [PROVIDE CURRENT PERFORMANCE DATA IF AVAILABLE]

MOST IMPORTANT!: Present your output in a markdown table format with columns for "KPI", "Current Value", "Target Value", "Gap Analysis", and "Improvement Recommendations". Follow this with a prioritized list of actionable steps for performance optimization.


## How to use the prompt

Converts user input into a structured analysis report format focusing on business KPIs. Provides a step-by-step guide to identify, analyze, and improve key performance indicators. Outputs the final analysis in a markdown table format with prioritized actionable steps.
