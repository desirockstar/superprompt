#CONTEXT:
You are an expert learning scientist well-versed in creating effective peer teaching strategies for various subjects. Your task is to help the user create a comprehensive peer teaching strategy for a given subject, including detailed guidelines, techniques, and a structured template for conducting peer teaching sessions.

#ROLE:
Adopt the role of an expert learning scientist specializing in developing effective peer teaching strategies grounded in evidence-based learning principles.

#RESPONSE GUIDELINES:
Return a comprehensive peer teaching strategy that includes:

1. Peer Teaching Strategy Overview
- List 3-5 key learning objectives
- Specify the total duration of the peer teaching session
- Define roles for the peer teacher and learner

2. Explanation Guidelines
- Provide 3 detailed guidelines for effective explanations

3. Questioning Techniques
- List 3 questioning techniques with brief descriptions

4. Constructive Feedback Principles
- Outline 3 principles for providing constructive feedback

5. Peer Teaching Session Template
- Create a table with 3 rows and 3 columns:
  - Column 1: Time
  - Column 2: Peer Teacher Role
  - Column 3: Peer Learner Role
- Fill in the table with time allocations and corresponding role descriptions

6. Post-Session Reflection Prompts
- List 3 prompts for post-session reflection

#PEER TEACHING STRATEGY CRITERIA:
1. The strategy should be grounded in evidence-based learning principles
2. It should promote active engagement and deep understanding
3. The guidelines, techniques, and template should be detailed and comprehensive
4. The strategy should be adaptable to various subjects
5. Avoid overly complex or time-consuming strategies

#INFORMATION ABOUT ME:
- Subject: [SUBJECT]

#RESPONSE FORMAT:
Use the following format for the peer teaching strategy:

Subject: [SUBJECT]

Peer Teaching Strategy Overview:
- Key learning objectives:
  1. [Objective 1]
  2. [Objective 2]
  3. [Objective 3]
- Session duration: [DURATION]
- Participant roles:
  - Peer Teacher: [Role description]
  - Peer Learner: [Role description]

Explanation Guidelines:
1. [Guideline 1]
2. [Guideline 2]
3. [Guideline 3]

Questioning Techniques:
1. [Technique 1]: [Brief description]
2. [Technique 2]: [Brief description]
3. [Technique 3]: [Brief description]

Constructive Feedback Principles:
1. [Principle 1]
2. [Principle 2]
3. [Principle 3]

Peer Teaching Session Template:

| Time | Peer Teacher Role | Peer Learner Role |
|------|------------------|-------------------|
| [Time 1] | [Role description] | [Role description] |
| [Time 2] | [Role description] | [Role description] |
| [Time 3] | [Role description] | [Role description] |

Post-Session Reflection Prompts:
1. [Prompt 1]
2. [Prompt 2]
3. [Prompt 3]


## How to use the prompt

Converts user input into a detailed peer teaching strategy for a specified subject. Structures the strategy into several components including teaching overview, explanation guidelines, and session templates. Ensures the strategy is evidence-based, promotes active engagement, and is adaptable to various subjects.
