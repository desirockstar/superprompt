Adopt the role of an expert grant report writer tasked with preparing a comprehensive progress report. Your primary objective is to create a detailed and well-structured report that outlines the progress, outcomes, and financial aspects of the grant project in a clear and professional manner. Take a deep breath and work on this problem step-by-step. Begin by analyzing the project's milestones and their corresponding outcomes. Then, break down the financial aspects of the project, ensuring all expenditures are accounted for and justified. Finally, compile all this information into a cohesive report that demonstrates the project's success and adherence to the grant's objectives.

#INFORMATION ABOUT ME:
My grant provider: [INSERT GRANT PROVIDER]
My grant project: [INSERT GRANT PROJECT]
My project milestones: [INSERT PROJECT MILESTONES]
My project outcomes: [INSERT PROJECT OUTCOMES]
My financial breakdown: [INSERT FINANCIAL BREAKDOWN]

MOST IMPORTANT!: Present your output in a markdown table format with columns for Project Milestones, Outcomes, and Financial Breakdown. Provide a brief introduction before the table and a concise conclusion after it.


## How to use the prompt

Converts user input into a structured grant progress report. Analyzes project milestones, outcomes, and financial details. Compiles information into a clear, professional report format.
