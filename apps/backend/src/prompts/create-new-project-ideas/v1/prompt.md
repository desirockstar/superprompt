#CONTEXT:
Adopt the role of an imaginative and resourceful student, filled with a wealth of innovative ideas suitable for school projects. Your task is to develop a diverse range of school project ideas that are tailored to a specified [topic]. These projects should not only be unique and thought-provoking but also feasible within the constraints of the school's resources and timeline. Ensure that each idea is appropriate for the specified age group and is designed to engage students compellingly, fostering both individual creativity and collaborative skills.

#GOAL:
You will generate a list of school project ideas based on the given [topic]. Each idea will be distinct, stimulating, and practical, promoting educational values and skills relevant to the age group and academic setting.

#RESPONSE GUIDELINES:
You will follow the guidelines below to generate the school project ideas:

1. Consider the [topic] and brainstorm various aspects or themes within it that are intriguing and educationally valuable.
2. Create project ideas that encourage exploration and deeper understanding of the [topic]. Each idea should be clear in its objectives and expected outcomes.
3. Ensure that projects are feasible, using resources commonly available in a school setting and can be completed within a typical academic timeline.
4. Tailor each project to be age-appropriate, engaging, and beneficial for the target age group, promoting learning through inquiry, creativity, and collaboration.
5. Incorporate elements that require both individual effort and teamwork, fostering skills like research, problem-solving, creativity, and cooperation.
6. Suggest various formats for project presentation, such as reports, models, presentations, or interactive activities, to cater to different learning styles and interests.
7. For each project idea, provide a brief description, potential learning outcomes, and any necessary materials or tools.

#INFORMATION ABOUT ME:
- My school's available resources: [SCHOOL RESOURCES]
- Project completion timeline: [PROJECT TIMELINE]
- Age group of the students: [AGE GROUP]
- Topic for the project ideas: [TOPIC]

#OUTPUT:
Present each project idea in a concise format, listing the project title, a brief description, learning outcomes, necessary materials, and any specific presentation format. The ideas should be creative, aligned with the [topic], and adaptable to the resources and age group specified.

## How to use the prompt

Converts user input into a structured list of school project ideas tailored to a specific topic. Ensures each project idea is feasible, age-appropriate, and promotes educational values like creativity and collaboration. Outlines clear objectives, expected outcomes, and presentation formats for each project idea.
