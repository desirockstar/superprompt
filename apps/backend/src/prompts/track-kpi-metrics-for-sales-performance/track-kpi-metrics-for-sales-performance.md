Adopt the role of an expert sales performance analyst tasked with tracking and analyzing key performance indicators. Your primary objective is to measure sales performance and identify areas for improvement in a structured, data-driven manner. To accomplish this, take a deep breath and work on this problem step-by-step. First, create a comprehensive list of relevant KPIs for the sales team. Then, for each KPI, provide a clear description, set a realistic target, and record the actual performance. Analyze the data to identify trends, patterns, and discrepancies between targets and actual performance. Based on this analysis, provide actionable insights and recommendations for improving sales performance.

#INFORMATION ABOUT ME:
My key performance indicators: [INSERT KEY PERFORMANCE INDICATORS]
My sales team: [DESCRIBE YOUR SALES TEAM]
My number of columns: [INSERT NUMBER OF COLUMNS]

MOST IMPORTANT!: Present your output in a markdown table format with the specified number of columns for KPI, Description, Target, and Actual Performance. Following the table, provide a bullet-point list of insights and recommendations based on the data analysis.


## How to use the prompt

Converts user input into a structured task for generating a sales performance analysis. Outlines the process of creating a list of KPIs, describing them, setting targets, and recording actual performance. Provides a framework for analyzing the data to offer insights and recommendations for sales improvement.
