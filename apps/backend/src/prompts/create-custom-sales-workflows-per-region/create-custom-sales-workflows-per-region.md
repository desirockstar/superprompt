#CONTEXT:
You are tasked with refining an existing sales workflow template to optimize sales processes across diverse geographical regions. The refined template should break down the workflow into key steps, define relevant performance metrics, highlight regional considerations, and assign implementation priorities.

#ROLE:
Adopt the role of a meticulous sales process engineer with deep expertise in designing optimized sales workflows across diverse geographical regions.

#RESPONSE GUIDELINES:
Develop a comprehensive table of sales workflow templates tailored for each geographical region. The table should have the following columns:
- Region
- Workflow Steps (use numbered steps to indicate sequence)
- Key Performance Metrics (use bullet points for multiple items)
- Regional Considerations (use bullet points for multiple items)
- Implementation Priority (use a scale of 1-5, with 5 being the highest priority)

Each cell in the table should contain clear and concise information, using bullet points for multiple items and numbered steps to indicate sequence where appropriate.

#TASK CRITERIA:
1. Ensure that the workflow steps are broken down into key stages that are relevant to each geographical region.
2. Define performance metrics that accurately reflect the success of the sales process in each region.
3. Highlight regional considerations that may impact the implementation of the sales workflow.
4. Assign implementation priorities based on the importance and urgency of optimizing the sales process in each region.
5. Use a clear and easy-to-read table format to present the information.

#INFORMATION ABOUT ME:
- My target regions: North America, Europe, Asia-Pacific, Latin America, Middle East and Africa
- My company's products/services: [DESCRIBE YOUR COMPANY'S PRODUCTS/SERVICES]
- My company's unique selling proposition: [DESCRIBE YOUR COMPANY'S UNIQUE SELLING PROPOSITION]
