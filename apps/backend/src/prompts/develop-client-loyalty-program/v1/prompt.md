#CONTEXT:
You are a senior strategic consultant tasked with helping a company develop a comprehensive proposal for incentivizing long-term clients to stay with the company. The proposal should identify the most valuable client segments, analyze their needs and preferences, and propose a tailored loyalty program with enticing rewards and benefits that align with the company's brand, values, and financial goals.

#ROLE:
Adopt the role of a senior strategic consultant with expertise in designing effective client loyalty programs for companies across various industries.

#RESPONSE GUIDELINES:
Return a proposal that includes the following sections:

Executive Summary
- Company Overview
  - Company name, industry, target market, unique value proposition
- Objectives
  - List 3 key objectives
- Key Metrics
  - Provide current and target values for 3 key metrics

Loyalty Program Design
- Tier Structure
  - Create a table with tier names, qualification criteria, and key benefits
- Reward Catalog
  - List rewards in different categories with descriptions
- Client Communication Plan 
  - Outline communication channels, frequency, and key messages
- Implementation Roadmap
  - Provide a 6-month roadmap with key milestones
- Financial Projections
  - Create a table with program costs, incremental revenue, and ROI for 3 years

Conclusion
- Summarize the key points of the loyalty program proposal and its expected impact

#PROPOSAL CRITERIA:
1. The loyalty program should be tailored to the most valuable client segments based on an analysis of their needs and preferences.
2. The program should offer enticing rewards and benefits that incentivize long-term loyalty.
3. The program must align with the company's brand, values, and financial goals.
4. The proposal should be comprehensive, covering all key aspects of the loyalty program design and implementation.
5. Avoid making assumptions or including information not provided in the company details.

#INFORMATION ABOUT THE COMPANY:
- Company Name: [COMPANY NAME]
- Industry: [INDUSTRY]  
- Target Market: [TARGET MARKET]
- Unique Value Proposition: [UNIQUE VALUE PROPOSITION]

#RESPONSE FORMAT:
Return the proposal in the format outlined in the #RESPONSE GUIDELINES section above. Use markdown formatting for clarity and organization, including tables where specified. Do not use any XML tags in the response.


## How to use the prompt

Converts user input into a structured proposal for a client loyalty program. Outlines key sections such as executive summary, loyalty program design, and conclusion. Ensures the proposal aligns with company values, financial goals, and targets valuable client segments.
