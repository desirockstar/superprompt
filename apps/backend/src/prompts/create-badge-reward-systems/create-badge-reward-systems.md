Adopt the role of an expert educational game designer tasked with creating a gamified learning system. Your primary objective is to develop an engaging and motivating badge reward system for a specific subject, tailored to a target audience. To accomplish this, take a deep breath and work on this problem step-by-step. First, analyze the subject matter and identify key learning objectives. Then, brainstorm creative and relevant badge ideas that align with these objectives. Next, establish clear achievement criteria for each badge, ensuring they are challenging yet attainable. Finally, determine appropriate rewards for each badge that will incentivize learners to progress through the system.

#INFORMATION ABOUT ME:
My subject: [INSERT SUBJECT]
My target audience: [INSERT TARGET AUDIENCE]
My number of columns: [INSERT NUMBER OF COLUMNS]

MOST IMPORTANT!: Present your output in a markdown table format with the specified number of columns. The columns should be titled [BADGE NAME], [ACHIEVEMENT CRITERIA], and [REWARD]. Provide at least 5 badge entries in your table, ensuring each entry is creative, relevant to the subject, and appropriately challenging for the target audience.


## How to use the prompt

Analyzes the subject matter to identify key learning objectives for the badge system. Brainstorms creative badge ideas that align with the established objectives. Establishes clear achievement criteria and determines appropriate rewards for each badge.
