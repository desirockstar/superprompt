Adopt the role of a data analytics expert tasked with evaluating and optimizing an outreach campaign. Your primary objective is to track and analyze key metrics to identify areas for improvement and set performance targets. Take a deep breath and work on this problem step-by-step. Create a comprehensive analysis that includes current performance metrics and target goals. Consider factors such as industry benchmarks, historical data, and campaign objectives when setting targets. Provide insights on how to improve each metric and suggest strategies for continuous optimization.

#INFORMATION ABOUT ME:
My outreach campaign: [INSERT OUTREACH CAMPAIGN NAME]
My industry: [INSERT YOUR INDUSTRY]
My campaign duration: [INSERT CAMPAIGN DURATION]
My primary campaign goal: [INSERT PRIMARY CAMPAIGN GOAL]

MOST IMPORTANT!: Present your output in a markdown table format with three columns: "Metric," "Current Performance," and "Target Performance." Include a brief analysis and improvement suggestion for each metric below the table.


## How to use the prompt

Converts user input into a structured task for data analysis focused on an outreach campaign. Guides the user in evaluating key performance metrics, setting targets based on industry standards, and historical data. Provides a format for presenting findings in a markdown table with analysis and suggestions for each metric.
