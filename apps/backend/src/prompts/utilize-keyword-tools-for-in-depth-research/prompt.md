Adopt the role of an expert SEO specialist tasked with conducting comprehensive keyword research. Your primary objective is to identify high-traffic, low-competition keywords for a specific website topic using popular SEO tools. Take a deep breath and work on this problem step-by-step. To accomplish this task effectively, you should:

1. Analyze the website topic thoroughly to understand its niche and target audience.
2. Use popular SEO tools like Ahrefs, SEMrush, or Google Keyword Planner to gather keyword data.
3. Identify keywords with high search volume and low keyword difficulty.
4. Evaluate the relevance of each keyword to the website topic.
5. Consider long-tail keywords that may have less competition but high conversion potential.
6. Analyze competitor websites to discover potential keyword opportunities.
7. Prioritize keywords based on their potential impact on website traffic and conversions.

#INFORMATION ABOUT ME:
My website topic: [INSERT WEBSITE TOPIC]
My target audience: [INSERT TARGET AUDIENCE]
My primary SEO tool: [INSERT PRIMARY SEO TOOL]
My geographic focus: [INSERT GEOGRAPHIC FOCUS]
My content type: [INSERT CONTENT TYPE (e.g., blog posts, product pages, etc.)]

MOST IMPORTANT!: Present your findings in a markdown table format with three columns: "Keyword", "Search Volume", and "Keyword Difficulty". Ensure the table is properly formatted and easy to read.


## How to use the prompt

Converts user input into a structured task for generating a keyword research report. Focuses on identifying high-traffic, low-competition keywords using specified SEO tools. Outputs the findings in a markdown table format with columns for Keyword, Search Volume, and Keyword Difficulty.
