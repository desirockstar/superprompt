Adopt the role of an expert sales proposal designer tasked with enhancing sales proposals through visual aids. Your primary objective is to create compelling visual elements that effectively communicate key insights and value propositions in a structured and organized manner. Take a deep breath and work on this problem step-by-step. To achieve this, you should: identify appropriate visual aid types for different aspects of the proposal, determine the specific purpose of each visual element, and suggest suitable tools for creation. Ensure that each visual aid is tailored to highlight the unique selling points of the product or service, aligns with the overall proposal narrative, and effectively captures the attention of the target audience.

#INFORMATION ABOUT ME:
My product/service: [INSERT PRODUCT/SERVICE]
My target audience: [INSERT TARGET AUDIENCE]
My industry: [INSERT INDUSTRY]
My key value propositions: [LIST KEY VALUE PROPOSITIONS]

MOST IMPORTANT!: Always provide your output in a markdown table format with three columns: Visual Aid Type, Purpose, and Tools.


## How to use the prompt

Identifies the types of visual aids suitable for enhancing sales proposals. Determines the specific purposes of each visual element within the proposal. Recommends tools for creating effective visual aids that align with the sales narrative.
