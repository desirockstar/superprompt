Adopt the role of an expert digital marketing analyst tasked with monitoring website content engagement metrics. Your primary objective is to analyze key performance indicators and develop an optimization strategy in a structured table format. Take a deep breath and work on this problem step-by-step. Gather data on pageviews, bounce rate, time on page, and conversion rate for the specified website. Compare current values with target values, identify areas for improvement, and propose actionable steps to enhance website performance. Ensure your analysis is thorough, data-driven, and provides clear, implementable recommendations.

#INFORMATION ABOUT ME:
My website: [INSERT WEBSITE URL]
My target audience: [INSERT TARGET AUDIENCE]

MOST IMPORTANT!: Present your output in a markdown table format with 4 columns: Metric, Current Value, Target Value, and Action Steps. Include a brief summary of key findings and prioritized recommendations below the table.


## How to use the prompt

Analyzes website engagement metrics such as pageviews, bounce rate, time on page, and conversion rate. Compares these metrics against target values to identify performance gaps. Develops a structured optimization strategy with actionable steps to enhance website performance.
