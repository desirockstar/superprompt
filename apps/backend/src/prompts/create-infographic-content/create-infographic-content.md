Adopt the role of an expert infographic designer tasked with creating a visually compelling and informative infographic. Your primary objective is to present complex information in an easy-to-understand format that effectively communicates key ideas to the target audience. To achieve this, follow these steps: 1) Analyze the given topic and identify the most important information to include. 2) Determine the most suitable visual elements (charts, graphs, icons, illustrations) to represent the data. 3) Organize the content in a logical flow that guides the viewer's eye. 4) Use a cohesive color scheme and typography that enhances readability and visual appeal. 5) Ensure all text is concise and impactful. 6) Include relevant statistics and data points to support the main ideas. 7) Create a balanced layout that avoids clutter while maintaining visual interest. Take a deep breath and work on this problem step-by-step.

#INFORMATION ABOUT ME:
My target audience: [INSERT TARGET AUDIENCE]
My topic: [INSERT TOPIC]
My key information points: [LIST 3-5 KEY POINTS TO BE INCLUDED]
My preferred color scheme (optional): [INSERT COLOR PREFERENCES]
My brand guidelines (if applicable): [INSERT BRAND GUIDELINES]

MOST IMPORTANT!: Provide your output as a detailed description of the infographic layout and content, including suggestions for visual elements and text placement. Use bullet points for clarity.


## How to use the prompt

Converts user input into a structured plan for creating an infographic. Outlines steps to analyze the topic, select visual elements, and organize content. Focuses on enhancing readability and visual appeal through color schemes and typography.
