Adopt the role of an expert SEO specialist tasked with optimizing meta tags site-wide for a website. Your primary objective is to improve SEO and drive organic traffic by crafting compelling meta titles and descriptions using the dependency grammar framework. Take a deep breath and work on this problem step-by-step. Analyze the website's content, target audience, and keywords to create meta tags that accurately represent each page while maximizing search engine visibility. Ensure that each meta title and description is unique, concise, and enticing to potential visitors. Provide a structured approach to crafting meta tags that balances keyword optimization with user engagement.

#INFORMATION ABOUT ME:
My website: [INSERT WEBSITE URL]
My target audience: [DESCRIBE TARGET AUDIENCE]
My primary keywords: [LIST TOP 5 KEYWORDS]
My industry: [SPECIFY INDUSTRY]
My unique selling proposition: [DESCRIBE UNIQUE SELLING PROPOSITION]

MOST IMPORTANT!: Provide your output in a markdown table format, with columns for Page URL, Meta Title, and Meta Description. Include examples for at least five different page types (e.g., homepage, product page, category page, blog post, about page).


## How to use the prompt

Analyzes the provided website content, target audience, and primary keywords to optimize meta tags. Crafts unique, concise, and compelling meta titles and descriptions for different page types using dependency grammar. Structures the output in a markdown table format with columns for Page URL, Meta Title, and Meta Description for at least five different page types.
