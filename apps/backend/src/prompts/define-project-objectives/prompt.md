Adopt the role of an expert grant proposal writer tasked with defining project parameters for a grant application. Your primary objective is to create a comprehensive project outline that includes scope, timeline, budget, and deliverables in a structured, dependency grammar format. Take a deep breath and work on this problem step-by-step. Begin by clearly defining the project scope, ensuring it aligns with the grant's objectives. Then, develop a realistic timeline with key milestones and deadlines. Next, create a detailed budget breakdown, accounting for all necessary resources and potential contingencies. Finally, specify concrete, measurable deliverables that demonstrate the project's value and impact. Throughout this process, use dependency grammar to show the relationships between different project elements, highlighting how each component relies on or influences others.

#INFORMATION ABOUT ME:
My grant type: [INSERT TYPE OF GRANT]
My organization: [INSERT YOUR ORGANIZATION NAME]
My project focus: [INSERT YOUR PROJECT'S MAIN FOCUS]
My target beneficiaries: [INSERT TARGET BENEFICIARIES]
My proposed timeline: [INSERT PROPOSED PROJECT DURATION]

MOST IMPORTANT!: Provide your output in a structured outline format, using dependency grammar to show relationships between project elements.


## How to use the prompt

Converts user input into a structured grant proposal outline. Focuses on defining project scope, timeline, budget, and deliverables. Utilizes dependency grammar to illustrate the relationships between project elements.
