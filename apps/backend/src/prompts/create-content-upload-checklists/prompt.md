##PHASE 4: Pre-Upload Verification Protocol
Your pre-upload checklist ensures content is publication-ready:

###Content Preparation Checklist:
□ Title optimized for SEO (includes primary keyword)
□ Meta description written (150-160 characters)
□ URL slug reviewed (no spaces, lowercase)
□ Primary category selected
□ Tags applied (3-5 relevant terms)
□ Featured image uploaded (correct dimensions)
□ Alt text added to all images
□ Internal links checked (minimum 2-3)
□ External links verified (no broken URLs)

###Quality Assurance:
□ Spell check completed
□ Grammar review passed
□ Readability score checked (target: 8th grade)
□ Fact-checking completed
□ Legal review if needed

Type "continue" for upload process checks.

##PHASE 5: Upload Process Verification
During upload, verify these CMS-specific elements:

###Technical Checklist:
□ Correct template selected
□ Publishing date/time confirmed
□ Author attribution accurate
□ Content permissions set correctly
□ Mobile preview checked
□ Desktop preview verified
□ Schema markup applied (if applicable)
□ Custom fields completed

###SEO Optimization:
□ Focus keyword in title
□ Keyword in first paragraph
□ Keyword density 1-2%
□ Related keywords included
□ Meta robots settings correct

Type "continue" for post-upload verification.

##PHASE 7: Accessibility Compliance Checklist
Ensure content meets accessibility standards:

###WCAG 2.1 Compliance:
□ Heading hierarchy logical (H1→H2→H3)
□ Color contrast ratio passes (4.5:1 minimum)
□ All images have descriptive alt text
□ Videos include captions
□ Audio content has transcripts
□ Links are descriptive (not "click here")
□ Tables have proper headers
□ Form fields labeled clearly
□ Keyboard navigation works
□ Screen reader compatible
