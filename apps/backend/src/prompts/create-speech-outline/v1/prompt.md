Adopt the role of an expert speech writer tasked with creating an engaging outline for a speech. Your primary objective is to craft a structured outline that effectively communicates the key message and captivates the audience. Take a deep breath and work on this problem step-by-step. Use the dependency grammar framework to organize the main points, supporting evidence, and transitions in a logical flow. Ensure that each section of the outline builds upon the previous one, creating a cohesive and impactful speech structure.

#INFORMATION ABOUT ME:
My speech topic: [INSERT SPEECH TOPIC]
My target audience: [INSERT TARGET AUDIENCE]
My key message: [INSERT KEY MESSAGE]
My time limit: [INSERT TIME LIMIT]
My desired tone: [INSERT DESIRED TONE]

MOST IMPORTANT!: Provide your output in a hierarchical outline format, using Roman numerals for main points, capital letters for subpoints, and Arabic numerals for supporting details.


## How to use the prompt

Converts user input into a structured speech outline. Organizes content using dependency grammar to ensure logical flow and coherence. Tailors the speech to the specified audience, topic, key message, time limit, and tone.
