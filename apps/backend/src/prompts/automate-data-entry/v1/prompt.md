Adopt the role of an expert automation specialist tasked with creating a comprehensive guide for data entry automation. Your primary objective is to develop a step-by-step instruction manual that streamlines the specified business process using the designated automation tool. To achieve this, you should: analyze the current data entry workflow, identify areas for improvement, design an efficient automation solution, and create clear, concise instructions with accompanying visuals. Ensure the guide is user-friendly, addresses potential challenges, and emphasizes accuracy throughout the automated process. Take a deep breath and work on this problem step-by-step.

#INFORMATION ABOUT ME:
My business process: [INSERT BUSINESS PROCESS]
My automation tool: [INSERT AUTOMATION TOOL]
My team's technical expertise: [INSERT LEVEL OF EXPERTISE]
My primary data sources: [INSERT DATA SOURCES]
My desired outcomes: [INSERT DESIRED OUTCOMES]

MOST IMPORTANT!: Provide your output as a numbered list with clear headings for each major step, and use bullet points for sub-steps or additional details. Include placeholders for screenshot descriptions where appropriate.


## How to use the prompt

Analyzes the existing data entry workflow to identify inefficiencies and areas for improvement. Designs an automation solution using the specified tool to streamline the data entry process. Creates a user-friendly instruction manual with step-by-step instructions, visuals, and addresses potential challenges to ensure accuracy.
