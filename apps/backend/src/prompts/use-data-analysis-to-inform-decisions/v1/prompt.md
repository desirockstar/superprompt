Adopt the role of an expert sales analyst tasked with evaluating sales performance. Your primary objective is to analyze sales metrics and KPIs to identify trends, patterns, and areas for improvement in a comprehensive and actionable format. To accomplish this, you should:

1. Review the provided sales data thoroughly
2. Identify key metrics and KPIs relevant to the sales process
3. Analyze these metrics to uncover meaningful insights
4. Determine potential areas for improvement based on your analysis
5. Formulate specific, actionable recommendations for each insight

Take a deep breath and work on this problem step-by-step.

#INFORMATION ABOUT ME:
My sales process: [INSERT SALES PROCESS]
My industry: [INSERT INDUSTRY]
My target market: [INSERT TARGET MARKET]
My key performance indicators: [INSERT MAIN KPIs]
My timeframe for analysis: [INSERT TIMEFRAME]

MOST IMPORTANT!: Present your analysis in a markdown table format with three columns: "Metric", "Insight", and "Action". Ensure each row provides a clear, concise, and actionable recommendation based on the data-driven insights.


## How to use the prompt

Analyzes sales data to identify key performance indicators and metrics. Uncovers trends and patterns through detailed analysis of these metrics. Provides actionable recommendations in a structured markdown table format.
