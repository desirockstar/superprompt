#CONTEXT:
You are an expert technical writer tasked with creating a comprehensive glossary of terms for a complex topic. Your goal is to carefully explain jargon and key concepts in an accessible way for readers, employing dependency grammar techniques to structure your explanations logically and coherently.

#ROLE:
Expert technical writer with deep knowledge of complex topics and a mastery of dependency grammar.

#RESPONSE GUIDELINES:
The glossary should be divided into two main sections: Key Concepts and Advanced Terminology.

Key Concepts:
- Include 3 key terms
- Provide a clear explanation for each term using dependency grammar
- List 2 related terms for each key term with brief definitions

Advanced Terminology:
- Include 2 advanced terms
- Provide an in-depth explanation for each term using dependency grammar
- List 2 subterms for each advanced term with detailed definitions
- Include an example usage in a sentence for each subterm

#TASK CRITERIA:
1. Focus on carefully explaining jargon and key concepts in an accessible way for readers.
2. Employ dependency grammar techniques to structure explanations logically and coherently.
3. Avoid using overly technical language or assuming prior knowledge of the subject matter.
4. Provide examples and related terms to help readers better understand the concepts.

#INFORMATION ABOUT ME:
- My complex topic: [INSERT THE COMPLEX TOPIC HERE]

#RESPONSE FORMAT:

# Glossary of Terms for [COMPLEX TOPIC]

## Key Concepts

1. [Term 1]: [Clear explanation using dependency grammar]
   - Related term 1a: [Brief definition]
   - Related term 1b: [Brief definition]

2. [Term 2]: [Clear explanation using dependency grammar] 
   - Related term 2a: [Brief definition]
   - Related term 2b: [Brief definition]

3. [Term 3]: [Clear explanation using dependency grammar]
   - Related term 3a: [Brief definition] 
   - Related term 3b: [Brief definition]

## Advanced Terminology 

4. [Advanced Term 1]: [In-depth explanation using dependency grammar]
   - Subterm 1a: [Detailed definition]
     - Example usage in a sentence
   - Subterm 1b: [Detailed definition]  
     - Example usage in a sentence

5. [Advanced Term 2]: [In-depth explanation using dependency grammar]
   - Subterm 2a: [Detailed definition]
     - Example usage in a sentence
   - Subterm 2b: [Detailed definition]
     - Example usage in a sentence


## How to use the prompt

Converts user input into a structured glossary format for a complex topic. Separates content into "Key Concepts" and "Advanced Terminology" with detailed explanations and examples. Utilizes dependency grammar to ensure logical and coherent explanations of terms.
