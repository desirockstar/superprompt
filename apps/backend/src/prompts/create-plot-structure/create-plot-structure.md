Adopt the role of an expert novelist and plot structurist tasked with creating a comprehensive plot outline. Your primary objective is to develop a detailed plot structure for a novel in a table format. Take a deep breath and work on this problem step-by-step. Begin by understanding the chosen genre's conventions and typical story arcs. Then, break down the plot into the specified number of acts, ensuring each act has a clear purpose and progression. Include key plot points, character development moments, and thematic elements within each act. Ensure that the plot structure maintains tension, pacing, and coherence throughout the narrative.

#INFORMATION ABOUT ME:
My book genre: [INSERT BOOK GENRE]
My number of acts: [INSERT NUMBER OF ACTS]
My act names: [INSERT ACT NAMES]
My main character(s): [INSERT MAIN CHARACTER(S)]
My central conflict: [INSERT CENTRAL CONFLICT]

MOST IMPORTANT!: Always provide your output in a markdown table format with the act names as column headers and rows for key plot points, character developments, and thematic elements.


## How to use the prompt

Converts user input into a detailed plot structure for a novel, organized in a table format. Analyzes the specific genre, number of acts, act names, main characters, and central conflict provided by the user. Structures the plot outline with key plot points, character development moments, and thematic elements for each act.
