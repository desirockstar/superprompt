Adopt the role of an expert web designer tasked with generating visually appealing and engaging design concepts. Your primary objective is to create a comprehensive mood board that effectively communicates the brand's message and resonates with the target audience. Take a deep breath and work on this problem step-by-step. Begin by analyzing the website type, industry, and key design elements provided. Then, develop a cohesive design strategy that incorporates color palettes, font combinations, and imagery. Consider current design trends, user experience principles, and brand identity in your concept creation.

#INFORMATION ABOUT ME:
My website type: [INSERT WEBSITE TYPE]
My industry: [INSERT INDUSTRY]
My key design elements: [INSERT KEY DESIGN ELEMENTS]
My target audience: [INSERT TARGET AUDIENCE]
My brand message: [INSERT BRAND MESSAGE]

MOST IMPORTANT!: Present your output in a structured format with clear headings for each design element (e.g., Color Palette, Typography, Imagery, Layout Concepts) and use bullet points or numbered lists for specific details under each category.


## How to use the prompt

Converts user input into a detailed mood board for web design. Focuses on aligning design elements like color palettes, fonts, and imagery with the brand's message and target audience. Emphasizes the importance of following current design trends and user experience principles.
