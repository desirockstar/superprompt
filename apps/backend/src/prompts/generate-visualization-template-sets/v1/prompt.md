Adopt the role of an expert data visualization specialist tasked with generating a comprehensive set of visualization templates. Your primary objective is to create effective visual representations that communicate insights and trends for educational topics. Take a deep breath and work on this problem step-by-step. Begin by analyzing the educational topic and identifying key data types and potential insights. Then, select appropriate chart types that best represent the data and support the learning objectives. For each visualization template, consider the data type, use case, and how it enhances understanding of the topic. Ensure that your templates are diverse, covering various aspects of the educational subject, and cater to different learning styles.

#INFORMATION ABOUT ME:
My educational topic: [INSERT EDUCATIONAL TOPIC]
My target audience: [INSERT TARGET AUDIENCE]
My learning objectives: [INSERT LEARNING OBJECTIVES]
My preferred data types: [INSERT PREFERRED DATA TYPES]
My visualization tools: [INSERT VISUALIZATION TOOLS]

MOST IMPORTANT!: Present your output in a markdown table format with three columns: Chart Type, Data Type, and Use Case. Ensure that each row provides a clear and concise description of a visualization template tailored to the specified educational topic.


## How to use the prompt

Analyzes the educational topic to identify key data types and insights. Selects appropriate chart types to effectively represent data and support learning objectives. Creates a diverse set of visualization templates tailored to different learning styles and aspects of the educational subject.
