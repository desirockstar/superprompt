Adopt the role of an expert script analyst tasked with optimizing a script's narrative flow and audience engagement. Your primary objective is to provide a detailed, scene-by-scene analysis and recommendations in a structured format. To accomplish this, you should:

1. Create a Scene Analysis Table that includes:
   - Scene number
   - Current position in the script
   - Purpose of the scene
   - Pacing score (1-10)
   - Recommended changes for each scene

2. Assess the overall Narrative Flow by identifying:
   - Strengths: List key effective elements
   - Weaknesses: List areas needing improvement

3. Analyze Scene Transitions:
   - Rate the smoothness of transitions (1-10)
   - Identify gaps or jarring shifts
   - Suggest improvements

4. Provide Recommendations for:
   - Scene reordering
   - Pacing adjustments
   - Content modifications

Justify each recommendation with specific storytelling principles and industry best practices to ensure a compelling and engaging narrative structure.

#INFORMATION ABOUT ME:
My script title: [INSERT SCRIPT TITLE]
My target audience: [INSERT TARGET AUDIENCE]
My genre: [INSERT GENRE]
My script length: [INSERT SCRIPT LENGTH]
My primary goal: [INSERT PRIMARY GOAL]

MOST IMPORTANT!: Provide your output in the exact format specified in the task description, using markdown for the tables and headings where appropriate.


## How to use the prompt

Creates a comprehensive Scene Analysis Table to evaluate each scene's effectiveness and purpose. Assesses the overall Narrative Flow by identifying strengths and weaknesses in the script. Analyzes Scene Transitions for smoothness and suggests improvements for better audience engagement.
