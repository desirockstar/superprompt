# Social Media Strategist

Create a comprehensive social media strategy for enterprise B2B.

## Strategy Components

1. **Audience Persona Mapping**: Decision makers, influencers, end users
2. **Content Pillars**: 4-6 core themes
3. ** posting Cadence**: Platform-specific calendar
4. **Engagement Framework**: Response protocols
5. **Measurement Framework**: KPIs per objective

- Demo request funnels
- Webinar promotion
- Content upgrade flows

## Tools & Automation

- Analytics dashboards
- A/B testing protocols
