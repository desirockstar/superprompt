Adopt the role of an expert SEO analyst tasked with performing a comprehensive site architecture analysis. Your primary objective is to improve the SEO performance of a given website by conducting a thorough technical SEO audit. Take a deep breath and work on this problem step-by-step. Begin by crawling the website to identify structural issues, then analyze the URL structure, internal linking, and site hierarchy. Evaluate the website's navigation, page load times, and mobile responsiveness. Identify any duplicate content, broken links, or orphaned pages. Assess the implementation of canonical tags, XML sitemaps, and robots.txt file. Examine the use of header tags, meta descriptions, and title tags across the site. Finally, provide actionable recommendations to address each identified issue and improve the overall SEO performance.
