Adopt the role of an expert SEO strategist tasked with developing a comprehensive programmatic SEO strategy. Your primary objective is to improve organic search visibility and drive targeted traffic for a specific website in a structured and actionable format. Take a deep breath and work on this problem step-by-step. Create a detailed strategy that includes effective tactics, relevant tools, and key performance indicators (KPIs) for each aspect of the programmatic SEO approach. Consider factors such as content optimization, technical SEO, link building, and user experience improvements. Provide insights on leveraging data-driven decisions and automating SEO processes where possible.

#INFORMATION ABOUT ME:
My website: [INSERT WEBSITE URL]
My target audience: [DESCRIBE TARGET AUDIENCE]
My industry: [SPECIFY INDUSTRY]
My main competitors: [LIST MAIN COMPETITORS]
My current SEO performance: [BRIEFLY DESCRIBE CURRENT SEO PERFORMANCE]

MOST IMPORTANT!: Present your output in a markdown table format with three columns: Tactics, Tools, and KPIs. Ensure each row provides a comprehensive strategy element with corresponding tools and measurable KPIs.


## How to use the prompt

Converts the user's detailed input into a structured SEO strategy. Focuses on creating actionable tactics, selecting relevant tools, and defining key performance indicators. Addresses specific aspects like content optimization, technical SEO, link building, and user experience.
