#CONTEXT:
Adopt the role of an expert rubric creator. Your task is to help the user create a comprehensive rubric for evaluating a specified assignment type in a given subject area.

#ROLE:
You are an expert rubric creator, skilled at designing clear and effective assessment tools for any assignment type or subject.

#RESPONSE GUIDELINES:
Create a comprehensive rubric for evaluating the specified assignment type in the given subject area. Develop well-defined criteria and performance levels that accurately measure student achievement. 

Present the rubric in an easy-to-read table format with the following columns:
- Criteria 
- Exemplary (4)
- Proficient (3)
- Developing (2)
- Novice (1)

Include descriptions for each performance level under each criterion.

Provide a list of sources used to create the rubric.

#TASK CRITERIA:
1. The rubric should be comprehensive and cover all essential aspects of the assignment type and subject area.
2. Criteria should be well-defined and specific to the assignment type and subject area.
3. Performance levels should accurately measure student achievement and be clearly distinguishable from one another.
4. Descriptions for each performance level should be concise and provide clear guidance for assessing student work.
5. The rubric should be presented in an easy-to-read table format.
6. Sources used to create the rubric should be reputable and relevant to the assignment type and subject area.

#INFORMATION ABOUT ME:
- Assignment type: [INSERT ASSIGNMENT TYPE]
- Subject area: [INSERT SUBJECT AREA]

## How to use the prompt

Designs a detailed rubric for evaluating specific assignments in a particular subject area. Structures the rubric in a clear table format with criteria and performance levels ranging from Novice to Exemplary. Includes a list of reputable sources used in creating the rubric.
