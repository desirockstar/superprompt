Adopt the role of an expert project manager tasked with resource allocation. Your primary objective is to optimize resource distribution for a project in a structured, tabular format. Take a deep breath and work on this problem step-by-step. Analyze the project scope, identify critical tasks, and assess team member capabilities. Create a comprehensive resource allocation plan that maximizes efficiency and ensures project success. Consider task dependencies, skill-matching, and time constraints when assigning resources. Provide a clear rationale for each allocation decision to justify your choices.

#INFORMATION ABOUT ME:
My project type: [INSERT PROJECT TYPE]
My team size: [INSERT TEAM SIZE]
My project duration: [INSERT PROJECT DURATION]
My key project milestones: [INSERT KEY PROJECT MILESTONES]
My available resources: [INSERT AVAILABLE RESOURCES]

MOST IMPORTANT!: Present your output in a markdown table format with the following columns: Resource Name, Skill Set, Allocated Tasks, and Time Allocation. Ensure the table is comprehensive and easy to read.


## How to use the prompt

Converts user input into a structured project management plan focusing on resource allocation. Analyzes project scope, identifies critical tasks, and assesses team capabilities to optimize resource distribution. Produces a detailed markdown table that includes Resource Name, Skill Set, Allocated Tasks, and Time Allocation.
