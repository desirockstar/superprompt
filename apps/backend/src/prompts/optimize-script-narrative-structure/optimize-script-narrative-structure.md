Adopt the role of an expert scriptwriter tasked with analyzing and optimizing a script's narrative structure and pacing. Your primary objective is to evaluate each scene and provide recommendations for improvement in a systematic, table-based format. To accomplish a well-structured and engaging script, you should assess the narrative purpose and pacing impact of each scene, suggest changes, and prioritize the improvements based on their importance to the overall story. Provide a comprehensive analysis that includes:

1. A scene analysis table with columns for scene number, narrative purpose, pacing impact (rated 1-10), suggested changes, and priority level (rated 1-5).

2. An assessment of the current scene flow, listing the existing scene order and transitions, followed by a recommended structure with proposed scene order and transitions.

3. A scene-by-scene evaluation highlighting strong elements (scenes that effectively advance the story, successful transitions, well-paced sequences) and areas for improvement (redundant scenes, pacing issues, weak transitions).

4. Recommendations for scene reordering, editing, rewriting, and transition improvements, providing rationale for each suggested change based on industry best practices and storytelling principles.

#INFORMATION ABOUT ME:
My script title: [INSERT SCRIPT TITLE]

My script length: [INSERT SCRIPT LENGTH]

MOST IMPORTANT!: Give your output in a markdown table format for the scene analysis and use bullet point lists for the other sections.


## How to use the prompt

Creates a detailed scene analysis table to evaluate narrative purpose and pacing impact. Provides a structured assessment of scene flow and recommendations for improvements. Offers a comprehensive scene-by-scene evaluation highlighting strengths and areas for enhancement.
