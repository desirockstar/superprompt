<context>
You are building an enterprise-grade resume builder that must compete with generic template generators flooding the market. Most resume builders fail because they produce one-size-fits-all documents that get filtered out by ATS systems or bore human reviewers. You need to create an intelligent web application that understands industry-specific hiring expectations - where a finance executive's resume should emphasize quantified achievements with conservative formatting, while a creative director needs visual hierarchy and project showcases. This isn't just about filling templates; it's about creating a professional tool that adapts design, content suggestions, and export optimization based on real-world hiring practices across different industries.
</context>

<role>
Adopt the role of an expert full-stack developer and UX designer who has built recruitment platforms for major companies and analyzed thousands of successful resumes across industries. Your primary objective is to create a complete, production-ready resume builder web application that dynamically generates industry-optimized resume designs in a comprehensive React TypeScript codebase with proper file structure and implementation details.
</role>

<output>
Structure your response with these sections:
● Complete Project Architecture and File Structure
● Industry-Specific Template Components with Distinct Designs
● Smart Form System with Conditional Logic
● Live Preview Engine with Real-Time Updates
● PDF Export System (ATS-Optimized and Visual Versions)
● Content Enhancement Tools and Industry-Specific Suggestions
● Professional UI Implementation with Responsive Design

Take a deep breath and work on this problem step-by-step. Provide complete, production-ready code with proper TypeScript interfaces, component implementations, and styling. Include inline documentation explaining design decisions and ensure the final application feels like a premium SaaS tool.
</output>

## How to use the prompt

Creates a dynamic resume builder tailored to industry-specific hiring practices. Generates resumes with adaptive design and content suggestions for different industries. Ensures resumes are ATS-optimized and visually appealing for human reviewers.
