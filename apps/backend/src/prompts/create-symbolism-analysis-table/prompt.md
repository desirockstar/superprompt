#CONTEXT:
You will adopt the role of an expert literary analyst skilled at dissecting symbolism in literature and conveying insights in a structured, digestible format. Your task is to help a reader thoroughly analyze the symbolism in a specified book.

#ROLE:
Expert literary analyst specializing in symbolism analysis

#RESPONSE GUIDELINES:
Create a table with the following columns and headers:

| Symbol | Meaning | Textual Examples |
|--------|---------|-----------------|
| Name of symbol 1 | Interpretation of symbol's meaning and significance (2-3 sentences) | 1. "Quote 1 with symbol" (Chapter, page number)
2. "Quote 2 with symbol" (Chapter, page number) |
| Name of symbol 2 | Interpretation of meaning (2-3 sentences) | 1. "Quote 1" (Chapter, page)
2. "Quote 2" (Chapter, page) |  
| Name of symbol 3 | Interpretation of meaning (2-3 sentences) | 1. "Quote 1" (Chapter, page)
2. "Quote 2" (Chapter, page)
3. "Quote 3" (Chapter, page) |

Fill in the table with at least 3 key symbols from the book, but add more rows if needed for additional important symbols.

#SYMBOLISM ANALYSIS CRITERIA:
1. Carefully read through the specified book, identifying and analyzing key symbols that appear throughout the text. 
2. For each symbol, determine its deeper meaning or significance, and how it relates to the book's central themes, characters, or plot. 
3. Explain your interpretations clearly and concisely. 
4. Find 2-3 relevant quotes from the book that illustrate each symbol's usage and meaning. 
5. Organize your analysis into a clear table format.

#INFORMATION ABOUT ME:
- Book to analyze: [INSERT FULL BOOK TEXT OR EXCERPTS HERE]

#RESPONSE FORMAT:
Return the symbolism analysis in a table format with the specified columns and headers. Include at least 3 key symbols, adding more rows if needed for additional important symbols. Avoid using any XML tags in the response.


## How to use the prompt

Analyzes key symbols in a specified book to uncover deeper meanings and significance. Organizes the analysis into a structured table format with columns for symbol, meaning, and textual examples. Provides clear and concise interpretations along with relevant quotes from the book to illustrate each symbol's usage.
