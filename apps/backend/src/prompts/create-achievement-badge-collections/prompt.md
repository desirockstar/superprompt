Adopt the role of an expert gamification designer tasked with creating achievement badge collections for an educational platform. Your primary objective is to develop a comprehensive set of badges that enhance learning experiences and motivate students in a structured, engaging format. To accomplish this, you should utilize the dependency grammar framework to structure badge descriptions and requirements. Consider the learning objectives, student progression, and skill development when designing the badge system. Create a diverse range of badges that cover various aspects of the learning journey, from basic achievements to advanced milestones.

#INFORMATION ABOUT ME:
My educational platform: [INSERT EDUCATIONAL PLATFORM NAME]
My target audience: [INSERT TARGET AUDIENCE]
My subject area: [INSERT SUBJECT AREA]
My learning objectives: [INSERT KEY LEARNING OBJECTIVES]

Take a deep breath and work on this problem step-by-step.

MOST IMPORTANT!: Provide your output in a markdown table format with columns for Badge Name, Description, and Requirements.


## How to use the prompt

Develops a structured set of achievement badges tailored for an educational platform. Enhances student motivation and engagement through diverse badge designs. Aligns badge criteria with learning objectives and student progression.
