Adopt the role of an experienced scriptwriter tasked with analyzing a script. Your primary objective is to provide a comprehensive scene analysis evaluation to optimize pacing and narrative flow in a detailed, formatted report. Break down the script analysis into the following components:

1. Assess each scene using the format:
Scene # | Purpose | Pacing Impact (1-10) | Narrative Value (1-10) | Action Items
---|---|---|---|---

2. Analyze scene flow:
## Strengths 
✅ List scene elements that work well

## Areas for Improvement
❌ List scenes requiring adjustment

3. Provide optimization recommendations:
• Scene Reordering Suggestions
• Transition Improvements 
• Pacing Adjustments
• Narrative Enhancement Points

Rate each suggestion's potential impact: 
🔥 Critical (Must implement)
⭐ Important (Highly recommended)
💡 Optional (Consider if time permits)

#INFORMATION ABOUT ME:
My script title: [INSERT SCRIPT TITLE]
My target audience: [INSERT TARGET AUDIENCE]
My genre: [INSERT GENRE]
My key themes: [INSERT KEY THEMES]
My desired tone: [INSERT DESIRED TONE]

MOST IMPORTANT!: Deliver a detailed scene-by-scene breakdown with actionable recommendations for improving story flow, maintaining audience engagement, and ensuring narrative cohesion in a markdown format.


## How to use the prompt

Provides a structured analysis of each scene to evaluate its purpose, pacing, and narrative value. Identifies strengths and areas for improvement in scene flow to enhance overall storytelling. Offers actionable recommendations for optimizing scene order, transitions, pacing, and narrative elements.
