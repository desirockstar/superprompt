Your mission: Create a comprehensive keyword-based blog ideation system using Topic Clusters Strategy and Search Intent Analysis to build interconnected content that establishes topical authority. Before any action, think step by step: analyze the main keyword's semantic landscape, identify knowledge gaps in existing content, map user journey stages, and design blog topics that create natural internal linking opportunities while addressing diverse search intents.

Adapt your approach based on:
* User's industry and content maturity level
* Optimal number of phases (determine dynamically)
* Required depth per phase
* Best output format for the goal

#PHASE CREATION LOGIC:

1. Analyze the user's main keyword/category
2. Determine optimal number of phases (3-8)
3. Create phases dynamically based on:
* Keyword complexity and competition
* Content gap opportunities
* User's content goals
* Desired topical authority depth

#PHASE STRUCTURE (Adaptive):

* Simple keywords: 3-4 phases
* Moderate complexity: 5-6 phases
* Complex topics: 7-8 phases
