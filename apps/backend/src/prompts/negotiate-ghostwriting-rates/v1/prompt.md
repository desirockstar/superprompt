Adopt the role of an expert negotiator tasked with facilitating rate discussions for a writing project. Your primary objective is to create a structured negotiation framework that balances project requirements with writer expertise in a clear, organized format. Take a deep breath and work on this problem step-by-step. Begin by analyzing the project scope and writer's experience level. Then, develop a comprehensive table that outlines project details, writer qualifications, and proposed rates. Use this table as a foundation to guide the negotiation process, ensuring fair compensation while maintaining project quality standards.

#INFORMATION ABOUT ME:
My writing project: [INSERT WRITING PROJECT]
My project scope: [INSERT PROJECT SCOPE]
My writer's experience level: [INSERT WRITER'S EXPERIENCE LEVEL]
My budget range (if applicable): [INSERT BUDGET RANGE]
My project timeline: [INSERT PROJECT TIMELINE]

MOST IMPORTANT!: Present your output in a markdown table format with three columns: Project Details, Writer's Experience, and Proposed Rates. Include a brief explanation for each proposed rate based on the project details and writer's experience.


## How to use the prompt

Converts user input into a structured negotiation framework for a writing project. Develops a comprehensive table that includes project details, writer qualifications, and proposed rates. Guides the negotiation process to ensure fair compensation and maintain project quality standards.
