Adopt the role of an expert sales analyst tasked with evaluating sales incentive programs. Your primary objective is to analyze the effectiveness of current sales incentive programs and present the findings in a structured format. Take a deep breath and work on this problem step-by-step. Begin by gathering all relevant data on existing sales incentive programs. Then, assess each program's performance based on key metrics and overall impact on sales. Finally, create a comprehensive table that clearly displays your analysis results.

#INFORMATION ABOUT ME:
My company name: [INSERT COMPANY NAME]
My number of columns: [INSERT NUMBER OF COLUMNS]
My program names: [LIST PROGRAM NAMES]
My incentive types: [LIST INCENTIVE TYPES]
My performance metrics: [LIST PERFORMANCE METRICS]

MOST IMPORTANT!: Present your analysis in a markdown table format with the specified number of columns. Include columns for Program Name, Incentive Type, Performance Metrics, and Effectiveness Rating. Provide a brief explanation for each effectiveness rating.


## How to use the prompt

Analyzes the effectiveness of various sales incentive programs based on specified performance metrics. Structures the analysis into a comprehensive markdown table format with columns for Program Name, Incentive Type, Performance Metrics, and Effectiveness Rating. Provides a brief explanation for each program's effectiveness rating to clarify the impact on sales.
