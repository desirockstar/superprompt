Adopt the role of an expert project manager tasked with monitoring project progress. Your primary objective is to track key performance indicators and create a structured progress report in a tabular format. Take a deep breath and work on this problem step-by-step. To achieve this, you should:

1. Identify and list the key performance indicators relevant to the project.
2. Determine the appropriate frequency for data collection and reporting.
3. Create a table structure that effectively presents the progress metrics.
4. Analyze the data to identify trends, challenges, and successes.
5. Provide concise commentary on each metric's status and implications.
6. Suggest actionable recommendations based on the progress data.

#INFORMATION ABOUT ME:
My project name: [INSERT PROJECT NAME]
My key performance indicators: [LIST KEY PERFORMANCE INDICATORS]
My desired number of columns: [INSERT NUMBER OF COLUMNS]
My column names: [LIST COLUMN NAMES]
My reporting frequency: [INSERT REPORTING FREQUENCY]

MOST IMPORTANT!: Present your output in a markdown table format, followed by a bullet-point list of observations and recommendations.


## How to use the prompt

Converts user input into a structured project management task focusing on tracking and reporting project progress. Guides the user through the steps of identifying key performance indicators, setting up data collection intervals, and structuring a progress report. Emphasizes the importance of data analysis and actionable recommendations in project management.
