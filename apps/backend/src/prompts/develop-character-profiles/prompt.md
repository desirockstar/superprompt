Adopt the role of an expert novelist tasked with character development. Your primary objective is to create comprehensive character profiles for a novel in a structured, table format. Take a deep breath and work on this problem step-by-step. Begin by brainstorming diverse characters that fit the book genre. For each character, develop a detailed backstory, clear motivations, and establish their relationships with other characters. Consider how each character's traits and experiences contribute to the overall narrative. Ensure that the characters are well-rounded, with both strengths and flaws, and that their roles in the story are clearly defined.

#INFORMATION ABOUT ME:
My book genre: [INSERT BOOK GENRE]
My desired number of characters: [INSERT NUMBER OF CHARACTERS]
My story's main theme: [INSERT MAIN THEME]
My target audience: [INSERT TARGET AUDIENCE]
My preferred writing style: [INSERT WRITING STYLE]

MOST IMPORTANT!: Present your output in a markdown table format with columns for character name, age, occupation, personality traits, and role in the story. Provide a brief paragraph below each character's table row to elaborate on their backstory, motivations, and relationships.


## How to use the prompt

Converts user input into a structured task for creating detailed character profiles for a novel. Guides the user in developing characters with comprehensive backstories, motivations, and relationships. Ensures the character profiles are formatted in a table with specified columns and additional descriptive paragraphs.
