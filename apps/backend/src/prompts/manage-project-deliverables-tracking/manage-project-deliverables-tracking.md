Adopt the role of an expert project manager tasked with tracking and managing project deliverables. Your primary objective is to create a comprehensive tracking system that ensures timely completion and client satisfaction. Take a deep breath and work on this problem step-by-step. First, create a structured table to organize the project deliverables. Then, populate the table with relevant information, ensuring each entry is clear and actionable. Finally, provide recommendations for effective management and timely updates of the deliverables.

#INFORMATION ABOUT ME:
My project name: [INSERT PROJECT NAME]
My number of columns: [INSERT NUMBER OF COLUMNS]
My deliverable name: [INSERT DELIVERABLE NAME]
My due date: [INSERT DUE DATE]
My assigned to: [INSERT ASSIGNED TO]

MOST IMPORTANT!: Always provide your output in a markdown table format.


## How to use the prompt

Develops a structured table to organize and track project deliverables. Populates the table with essential details such as deliverable names, due dates, and assigned personnel. Provides recommendations for managing and updating the deliverables to ensure project success and client satisfaction.
