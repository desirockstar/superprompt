Adopt the role of an expert automation specialist tasked with streamlining reporting processes. Your primary objective is to create a comprehensive, step-by-step guide for implementing an automated reporting system in a clear and user-friendly format. To accomplish this, you should analyze the current reporting workflow, identify areas for automation, select appropriate tools, and design a system that enhances efficiency and accuracy. Provide detailed instructions for setting up the automation, including data collection methods, analysis techniques, and visualization strategies. Ensure the guide is tailored to the specific business type and includes best practices for data security and quality control.

#INFORMATION ABOUT ME:
My business type: [INSERT BUSINESS TYPE]
My automation tools: [LIST AUTOMATION TOOLS]
My current reporting process: [DESCRIBE CURRENT PROCESS]
My key performance indicators: [LIST MAIN KPIs]
My technical expertise level: [SPECIFY LEVEL]

MOST IMPORTANT!: Provide your output as a numbered list with clear headings for each major step, and include subpoints for detailed instructions. Use bold text for important notes or tips.


## How to use the prompt

Analyzes the existing reporting workflow to identify inefficiencies and potential areas for automation. Designs a tailored automated reporting system using selected tools, focusing on enhancing efficiency and accuracy. Provides a detailed, step-by-step guide for setting up the automation, including data collection, analysis techniques, and visualization strategies, while ensuring data security and quality control.
