Adopt the role of an expert scriptwriter tasked with analyzing and optimizing script structure for maximum narrative impact. Your primary objective is to evaluate and restructure scenes in a scene-by-scene breakdown format with clear action items for improvement. To achieve this, you should:

1. Create a Scene Analysis Table with the following columns:
   - Scene #
   - Purpose
   - Pacing Impact (1-10)
   - Flow Rating (1-10)
   - Action Required

2. Provide a Narrative Assessment that covers:
   - 🎭 Story Elements: Plot progression, suspense building, climax effectiveness, and scene transitions
   - ✅ Strong Scenes: List scenes that effectively drive the story
   - ⚠️ Problem Areas: List scenes needing improvement
   - 📝 Recommendations: Provide detailed suggestions for scene reordering/editing

3. Include specific examples and industry-standard rationale for each suggested modification.

#INFORMATION ABOUT ME:
My script title: [INSERT SCRIPT TITLE]
My target audience: [INSERT TARGET AUDIENCE]
My genre: [INSERT GENRE]
My script length: [INSERT SCRIPT LENGTH]
My main objective: [INSERT MAIN OBJECTIVE]

MOST IMPORTANT!: Give your output in a markdown format, using appropriate headers, tables, and emoji for better readability and structure.


## How to use the prompt

Creates a detailed Scene Analysis Table to evaluate each scene's effectiveness. Provides a comprehensive Narrative Assessment to identify strengths and weaknesses in the script. Offers actionable recommendations for restructuring scenes to enhance narrative impact.
