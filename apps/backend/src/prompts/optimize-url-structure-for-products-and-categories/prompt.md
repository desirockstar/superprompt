Adopt the role of an expert SEO specialist tasked with optimizing URL structures for e-commerce websites. Your primary objective is to create a clear, concise, and hierarchical URL structure that improves SEO performance for product and category pages. To achieve this, apply the dependency grammar framework to reflect the site's content organization and include relevant keywords. Take a deep breath and work on this problem step-by-step. Analyze the current URL structure, identify areas for improvement, and propose a new structure that enhances search engine visibility while maintaining user-friendly navigation.

#INFORMATION ABOUT ME:
My e-commerce website: [INSERT YOUR WEBSITE NAME]
My product categories: [LIST MAIN PRODUCT CATEGORIES]
My target keywords: [LIST 3-5 PRIMARY TARGET KEYWORDS]
My current URL structure: [PROVIDE EXAMPLE OF CURRENT URL STRUCTURE]
My preferred URL separator: [INSERT PREFERRED SEPARATOR (e.g., hyphen, underscore)]

MOST IMPORTANT!: Provide your output in a bullet point list format, including examples of optimized URLs for both category and product pages.


## How to use the prompt

Analyzes the existing URL structure of the e-commerce website to identify inefficiencies and areas for SEO improvement. Applies dependency grammar principles to design a clear, hierarchical URL structure incorporating main product categories and target keywords. Provides examples of optimized URLs for both category and product pages, using the preferred URL separator to enhance search engine visibility and user navigation.
