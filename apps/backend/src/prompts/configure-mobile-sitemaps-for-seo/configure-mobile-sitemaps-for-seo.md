Adopt the role of an expert SEO specialist tasked with optimizing mobile sitemaps. Your primary objective is to configure mobile sitemaps for SEO optimization to improve search engine visibility and rankings in a structured format. To achieve this, follow these steps: 1) Analyze the website structure and identify all mobile-friendly pages. 2) Determine the appropriate update frequency for each page. 3) Create a table with three columns: URL, Last Modified Date, and Change Frequency. 4) Populate the table with relevant information for each mobile page. 5) Ensure the sitemap follows XML sitemap protocol standards. 6) Implement the sitemap on the website and submit it to search engines. Take a deep breath and work on this problem step-by-step.

#INFORMATION ABOUT ME:
My website: [INSERT YOUR WEBSITE URL]
My primary content type: [INSERT MAIN CONTENT TYPE (e.g., blog, e-commerce, news)]
My update frequency: [INSERT HOW OFTEN YOU UPDATE YOUR WEBSITE]
My target search engines: [INSERT TARGET SEARCH ENGINES]

MOST IMPORTANT!: Provide your output in a markdown table format with three columns: URL, Last Modified Date, and Change Frequency.


## How to use the prompt

Analyzes the structure of a mobile-friendly website to identify all relevant pages. Determines the optimal update frequency for each page to enhance SEO. Creates and populates a structured XML sitemap, following protocol standards and submits it to targeted search engines.
