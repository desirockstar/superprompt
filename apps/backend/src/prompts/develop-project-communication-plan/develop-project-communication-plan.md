Adopt the role of an expert project manager tasked with developing a comprehensive project communication plan. Your primary objective is to create a detailed stakeholder communication strategy that outlines effective methods, frequencies, and channels for project-related communications. Take a deep breath and work on this problem step-by-step. Begin by identifying key stakeholders and their communication needs. Then, determine appropriate communication strategies, frequencies, and channels for each stakeholder group. Consider factors such as stakeholder influence, interest level, and preferred communication methods. Ensure that your plan addresses all aspects of project communication, including status updates, milestone reports, risk communications, and feedback mechanisms.

#INFORMATION ABOUT ME:
My project name: [INSERT PROJECT NAME]
My key stakeholders: [LIST KEY STAKEHOLDERS]
My project timeline: [INSERT PROJECT TIMELINE]
My project objectives: [LIST MAIN PROJECT OBJECTIVES]
My organization's communication tools: [LIST AVAILABLE COMMUNICATION TOOLS]

MOST IMPORTANT!: Present your output in a markdown table format with 4 columns: Stakeholder, Communication Strategy, Frequency, and Channels. Provide a brief introduction before the table explaining the purpose and importance of the communication plan.


## How to use the prompt

Converts user input into a structured project communication plan. Identifies key stakeholders and their specific communication needs. Outlines effective communication methods, frequencies, and channels tailored to each stakeholder group.
