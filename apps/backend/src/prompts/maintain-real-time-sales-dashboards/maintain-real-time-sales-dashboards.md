Adopt the role of an expert data visualization specialist tasked with creating and maintaining real-time sales dashboards. Your primary objective is to design an effective dashboard that tracks key performance indicators (KPIs) for a sales operations team. Take a deep breath and work on this problem step-by-step. First, identify the most relevant KPIs for the sales team. Then, create a structured table layout that clearly presents each KPI, its current value, target value, and variance. Ensure the dashboard is visually appealing, easy to read, and provides actionable insights at a glance. Include recommendations for data refresh rates, color coding for performance indicators, and any additional features that would enhance the dashboard's utility.

#INFORMATION ABOUT ME:
My sales operations team: [INSERT SALES OPERATIONS TEAM]
My key performance indicators: [LIST KEY PERFORMANCE INDICATORS]
My number of columns: [INSERT NUMBER OF COLUMNS]

MOST IMPORTANT!: Provide your output in a markdown table format, showcasing the structure of the dashboard with example data.


## How to use the prompt

Identifies and prioritizes the most relevant KPIs for the sales team. Designs a structured table layout for the dashboard that includes KPIs, current values, target values, and variances. Recommends visual enhancements and features like data refresh rates and color coding to improve dashboard utility and readability.
