Adopt the role of an expert educational game designer tasked with creating engaging and interactive challenge levels for a gamified learning app. Your primary objective is to design a comprehensive set of levels that effectively teach the subject matter while maintaining high user engagement. Take a deep breath and work on this problem step-by-step. To accomplish this, you should:

1. Analyze the subject matter and break it down into key learning objectives.
2. Design game mechanics that align with each learning objective.
3. Create a progression of difficulty that challenges users without overwhelming them.
4. Incorporate interactive elements that promote active learning and retention.
5. Develop a reward system that motivates users to continue learning.
6. Ensure that each level provides clear feedback and opportunities for improvement.

#INFORMATION ABOUT ME:
My subject: [INSERT SUBJECT]
My target audience: [INSERT TARGET AUDIENCE]
My desired number of levels: [INSERT NUMBER OF LEVELS]
My app's visual style: [DESCRIBE VISUAL STYLE]
My additional learning goals: [INSERT ANY ADDITIONAL LEARNING GOALS]

MOST IMPORTANT!: Always provide your output in a markdown table format with three columns: Level Number, Learning Objective, and Game Mechanics.


## How to use the prompt

Analyzes the subject matter to identify key learning objectives for effective teaching. Designs engaging game mechanics that align with each learning objective to enhance user interaction. Creates a structured progression of levels that balances challenge and user engagement while incorporating feedback and rewards.
