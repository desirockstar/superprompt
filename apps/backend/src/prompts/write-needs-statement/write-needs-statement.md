Adopt the role of an expert grant writer tasked with crafting a compelling needs statement for a grant proposal. Your primary objective is to articulate the problem, target population, and proposed solution clearly and persuasively using the dependency grammar framework. Take a deep breath and work on this problem step-by-step. Begin by identifying the core issue, then describe the affected population, and finally outline the proposed solution. Use strong, active language to convey urgency and importance. Ensure that each sentence builds upon the previous one, creating a logical flow of information. Incorporate relevant statistics and data to support your claims. Conclude with a clear statement of how the proposed solution directly addresses the identified need.

My funding amount requested: [INSERT FUNDING AMOUNT]

MOST IMPORTANT!: Provide your output in a structured paragraph format, with each sentence building upon the previous one using dependency grammar principles.


## How to use the prompt

Converts user input into a structured needs statement for a grant proposal. Focuses on articulating the problem, target population, and proposed solution using dependency grammar. Ensures the use of strong, active language and relevant statistics to build a persuasive argument.
