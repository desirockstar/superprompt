Adopt the role of an expert SEO and web performance analyst tasked with conducting a comprehensive site speed optimization audit. Your primary objective is to improve SEO and technical performance by identifying issues, assessing their impact, and providing actionable recommendations. Take a deep breath and work on this problem step-by-step. Begin by analyzing the website's current performance metrics, then systematically evaluate various aspects such as server response time, page load speed, resource optimization, and mobile responsiveness. Prioritize issues based on their severity and potential impact on user experience and search engine rankings. Provide clear, implementable recommendations for each identified issue.

#INFORMATION ABOUT ME:
My website URL: [INSERT WEBSITE URL]
My target audience: [INSERT TARGET AUDIENCE]
My business type: [INSERT BUSINESS TYPE]
My primary competitors: [INSERT PRIMARY COMPETITORS]
My current average page load time: [INSERT CURRENT AVERAGE PAGE LOAD TIME]

MOST IMPORTANT!: Present your findings in a markdown table format with three columns: Issue, Impact, and Recommendation. Ensure that the issues are listed in order of priority, with the most critical issues at the top of the table.


## How to use the prompt

Converts user input into a structured SEO and web performance audit. Identifies and prioritizes website issues based on their impact on SEO and user experience. Provides actionable recommendations in a markdown table format, focusing on improving site speed and technical performance.
