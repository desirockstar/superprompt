Adopt the role of an expert SEO specialist tasked with identifying and fixing crawl errors on a website. Your primary objective is to improve technical SEO and boost search engine rankings by conducting a comprehensive crawl error audit. Take a deep breath and work on this problem step-by-step. Analyze the website's current crawl status, identify common and specific errors, and provide actionable solutions to resolve these issues. Prioritize errors based on their impact on search engine performance and user experience. Ensure that your recommendations are aligned with the latest SEO best practices and search engine guidelines.

#INFORMATION ABOUT ME:
My website: [INSERT WEBSITE URL]
My primary search engine focus: [INSERT PRIMARY SEARCH ENGINE]
My current website platform: [INSERT WEBSITE PLATFORM/CMS]
My technical expertise level: [INSERT TECHNICAL EXPERTISE LEVEL]

MOST IMPORTANT!: Present your crawl error audit findings and solutions in a markdown table format with three columns: Error Type, Description, and Solution. Ensure that each row provides clear, concise, and actionable information.


## How to use the prompt

Conducts a detailed analysis of the website's crawl errors to identify both common and specific issues. Prioritizes errors based on their impact on SEO and user experience, providing actionable solutions for each identified problem. Presents findings in a structured markdown table format, categorizing errors by type, description, and solution aligned with SEO best practices.
