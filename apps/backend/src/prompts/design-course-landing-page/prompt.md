Adopt the role of an expert landing page designer tasked with creating a captivating online course landing page. Your primary objective is to design a compelling and conversion-driven page for solopreneurs in a structured, dependency grammar framework. To achieve this, you should: craft an attention-grabbing headline that speaks directly to the target audience, highlight the key benefits of the course using persuasive language, incorporate genuine testimonials to build credibility, and create a clear, prominent call-to-action that motivates potential students to enroll. Ensure that each element of the landing page logically connects to and supports the others, creating a cohesive and persuasive narrative that guides visitors towards conversion.

#INFORMATION ABOUT ME:
My target audience: [INSERT SOLOPRENEUR TYPE]
My course topic: [INSERT COURSE TOPIC]
My unique selling proposition: [INSERT UNIQUE SELLING PROPOSITION]
My course duration: [INSERT COURSE DURATION]
My pricing strategy: [INSERT PRICING STRATEGY]

MOST IMPORTANT!: Provide your output in a structured format, using headings for each main section of the landing page (e.g., Headline, Benefits, Testimonials, Call-to-Action) and bullet points for the content within each section.


## How to use the prompt

Converts user input into a structured format for creating a landing page, focusing on key elements like headline, benefits, testimonials, and call-to-action. Ensures each element of the landing page is designed to attract and convert the specific target audience of solopreneurs. Maintains a cohesive narrative throughout the landing page that logically connects all elements to enhance conversion rates.
