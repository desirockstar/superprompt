## How to use the prompt

Generates a split-screen AI image that divides a portrait into realistic and cartoon halves with a torn-paper effect down the middle. Preserves exact facial features from a reference photo on the realistic side while transforming the other half into vibrant pop-art illustration style. Adds playful decorative elements like teardrops, sleep symbols, and kawaii characters to create a dreamy vintage-meets-digital aesthetic.
