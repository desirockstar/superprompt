Adopt the role of an expert SEO specialist tasked with automating redirect management. Your primary objective is to create a comprehensive system for managing redirects to improve SEO and streamline programmatic SEO processes. Take a deep breath and work on this problem step-by-step. Create a detailed process that includes setting up a redirect management table, implementing redirects, monitoring their effectiveness, and updating as necessary. Consider factors such as redirect types, impact on search engine rankings, user experience, and site performance. Provide clear instructions on how to maintain and update the redirect system over time.

#INFORMATION ABOUT ME:
My website: [INSERT YOUR WEBSITE URL]
My number of columns: [INSERT NUMBER OF COLUMNS]
My redirect types: [LIST YOUR REDIRECT TYPES]
My current SEO ranking: [INSERT YOUR CURRENT SEO RANKING]
My primary SEO goals: [DESCRIBE YOUR PRIMARY SEO GOALS]

MOST IMPORTANT!: Present your output in a markdown table format with the specified number of columns for Redirect Type, Source URL, Target URL, and Status. Include detailed instructions for implementation and maintenance.


## How to use the prompt

Converts the user input into a structured guide for setting up and managing an automated redirect system to enhance SEO. Details the process of creating a redirect management table, implementing redirects, monitoring their effectiveness, and updating as necessary. Focuses on redirect types, their impact on SEO, user experience, and site performance, along with maintenance instructions.
