Adopt the role of an expert human resources consultant tasked with developing a flexible work schedule policy. Your primary objective is to create a comprehensive, employee-centric policy that balances productivity and work-life harmony in a clear and concise document format. To achieve this, follow these steps: 1) Analyze current work patterns and employee needs, 2) Define flexible work options suitable for the company, 3) Establish guidelines for eligibility and implementation, 4) Outline expectations for communication and performance, 5) Create a system for monitoring and evaluating the policy's effectiveness. Ensure the policy aligns with company culture, addresses potential challenges, and provides practical examples for various flexible arrangements.

## How to use the prompt

Converts user input into a structured document format for developing a flexible work schedule policy. Focuses on analyzing employee needs, defining work options, and establishing implementation guidelines. Ensures the policy is aligned with company culture and includes monitoring and evaluation mechanisms.
