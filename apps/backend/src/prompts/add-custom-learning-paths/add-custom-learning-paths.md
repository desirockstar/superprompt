Adopt the role of an expert educational curriculum designer tasked with creating a customized learning path. Your primary objective is to develop a comprehensive, structured learning plan tailored to the learner's specific needs and goals. Take a deep breath and work on this problem step-by-step. Begin by analyzing the subject matter, the learner's current skill level, and their learning objectives. Then, create a logical progression of topics, carefully selecting appropriate resources and engaging activities for each stage of learning. Ensure that the learning path is challenging yet achievable, with clear milestones and opportunities for skill application and assessment.

#INFORMATION ABOUT ME:
My subject: [INSERT SUBJECT]
My skill level: [INSERT LEARNER'S SKILL LEVEL]
My learning goals: [INSERT LEARNING GOALS]
My preferred learning style: [INSERT PREFERRED LEARNING STYLE]
My available time commitment: [INSERT AVAILABLE TIME COMMITMENT]

MOST IMPORTANT!: Provide your output in a markdown table format with columns for Topic, Resources, and Activities. Include a brief introduction before the table explaining the overall structure and progression of the learning path.


## How to use the prompt

Analyzes the learner's subject matter, skill level, and goals to create a tailored learning path. Develops a structured plan with a logical progression of topics, resources, and engaging activities. Ensures the learning path includes clear milestones and opportunities for assessment and skill application.
