#CONTEXT:
You are acting as a spreadsheet wizard tasked with designing a comprehensive table to efficiently track partner incentives. The table should capture essential information for monitoring and analyzing the effectiveness of various incentive programs, providing a clear, well-structured format that facilitates easy data entry and interpretation.

#ROLE:
Adopt the role of a spreadsheet wizard with expertise in creating effective data tracking solutions tailored to specific business needs.

#RESPONSE GUIDELINES:
Create a table with the following columns:

1. Partner Name: Enter the name of the partner organization or individual.

2. Incentive Type: Specify the type of incentive offered (e.g., discount, rebate, promotional item).

3. Start Date: Record the date the incentive program begins (format: YYYY-MM-DD).

4. End Date: Record the date the incentive program ends (format: YYYY-MM-DD).

5. Results: Quantify the impact of the incentive program using the following metrics:
   - Sales volume: [Enter total sales volume generated during the incentive period]
   - New customers acquired: [Enter number of new customers acquired during the incentive period]
   - Revenue growth: [Enter percentage increase in revenue compared to the previous period]

Use conditional formatting to highlight the following:
- Start Date: Highlight in blue if the start date is within the next 30 days.
- End Date: Highlight in red if the end date is within the next 7 days or has already passed.
- Results: Highlight in green if the incentive program meets or exceeds the following thresholds:
  - Sales volume: [Enter target sales volume]
  - New customers acquired: [Enter target number of new customers]
  - Revenue growth: [Enter target percentage increase]

#TASK CRITERIA:
1. The table should be comprehensive and capture all essential information for tracking partner incentives effectively.
2. The format should be clear, well-structured, and easy to understand.
3. The table should facilitate easy data entry and interpretation.
4. Conditional formatting should be used to highlight important dates and results.
5. Avoid unnecessary complexity or extraneous information that may clutter the table.

#INFORMATION ABOUT ME:
- My target sales volume: [INSERT TARGET SALES VOLUME]
- My target number of new customers: [INSERT TARGET NUMBER OF NEW CUSTOMERS]
- My target revenue growth percentage: [INSERT TARGET REVENUE GROWTH PERCENTAGE]

#RESPONSE FORMAT:
The response should be formatted as a table with the specified columns and conditional formatting rules. No additional formatting or explanations should be included.


## Tips for this prompt

Designs a structured and comprehensive table for tracking partner incentives. Implements conditional formatting to highlight key dates and results based on predefined criteria. Ensures the table is clear and facilitates easy data entry and interpretation.

## How to use the prompt

Design the spreadsheet with dropdown menus for the "Incentive Type" column to standardize entries and minimize data entry errors, ensuring consistency across the dataset. Implement data validation rules to restrict date entries in the "Start Date" and "End Date" columns, ensuring all dates are entered in the correct format (YYYY-MM-DD) and are logical (e.g., End Date must be after Start Date). Set up automatic alerts or email notifications linked to the spreadsheet that inform relevant team members when an incentive's end date is approaching or when specific targets (like sales volume or revenue growth) are met or exceeded.
