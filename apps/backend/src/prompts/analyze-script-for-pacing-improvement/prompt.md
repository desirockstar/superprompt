Adopt the role of an expert scriptwriter tasked with analyzing a script. Your primary objective is to provide a comprehensive scene-by-scene breakdown to optimize pacing and narrative flow in a structured, formatted analysis report. To accomplish this, you should evaluate each scene's contribution to the story, assess the overall scene structure and flow, and provide specific improvement recommendations. Create a detailed analysis that enhances the script's engagement and narrative impact.

#INFORMATION ABOUT ME:
My script title: [INSERT SCRIPT TITLE]
My target audience: [INSERT TARGET AUDIENCE]
My genre: [INSERT GENRE]
My script length: [INSERT SCRIPT LENGTH]

MOST IMPORTANT!: Format your analysis report using the following structure:

## Scene Structure Analysis 
📝 For each scene provide:
- Scene Number/Name 
- Current Timeline Position
- Purpose in Story
- Pacing Impact (Fast ⚡ / Medium 🏃 / Slow 🐌)  
- Engagement Score (1-10)

## Scene Flow Assessment
✅ Scenes that work well
❌ Scenes needing adjustment  
⚠️ Pacing concerns
↕️ Suggested scene reordering

## Improvement Recommendations
For each scene flagged for improvement:
- Current Issue
- Proposed Change
- Expected Impact 
- Transition Notes

## Overall Narrative Flow Score
🎯 Pacing: [1-10]
🎬 Structure: [1-10] 
📈 Engagement: [1-10]
🔄 Scene Transitions: [1-10]


## How to use the prompt

Provides a detailed scene-by-scene breakdown to enhance pacing and narrative flow. Evaluates each scene's contribution to the story and suggests specific improvements. Delivers a structured analysis report that increases engagement and narrative impact.
