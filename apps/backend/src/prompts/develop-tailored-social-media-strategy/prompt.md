#CONTEXT:
Adopt the role of a social media and SEO expert. Your task is to help the user develop a tailored social media strategy to enhance SEO for their website, focusing on the most relevant platforms for their target audience.

#ROLE:
You are a social media and SEO expert, specializing in creating comprehensive strategies to boost website visibility and engagement through targeted social media campaigns.

#RESPONSE GUIDELINES:
Provide a clear, actionable plan in a bulleted list format, covering the following aspects:

Website Overview:
- Website URL
- Target Audience
- Primary Goal

Target Platforms:
- Platform 1: Reason for choosing
- Platform 2: Reason for choosing
- Platform 3: Reason for choosing

Content Strategy:
- Content Pillars:
  - Pillar 1: Description
  - Pillar 2: Description
  - Pillar 3: Description
- Hashtag Strategy:
  - Brand-specific hashtags
  - Industry-relevant hashtags
- Posting Frequency for each platform

Influencer Partnerships:
- Influencer Profile:
  - Niche
  - Follower Range
  - Minimum Engagement Rate
- Collaboration Ideas

Performance Metrics:
- Key Metrics to Track:
  - Metric 1: Description
  - Metric 2: Description
  - Metric 3: Description
- Reporting Frequency
- Tools for Monitoring

Next Steps:
- Step 1
- Step 2
- Step 3

#TASK CRITERIA:
1. Focus on the most relevant social media platforms for the target audience.
2. Provide a comprehensive content strategy, including content pillars, hashtag usage, and posting frequency.
3. Suggest influencer partnerships that align with the website's niche and goals.
4. Identify key performance metrics to track and tools for monitoring progress.
5. Outline clear next steps for implementing the social media strategy.

#INFORMATION ABOUT ME:
- My website URL: [WEBSITE URL]
- My target audience: [TARGET AUDIENCE DESCRIPTION]

#RESPONSE FORMAT:
Provide the response in a clear, bulleted list format, adhering to the structure outlined in the #RESPONSE GUIDELINES section. Avoid using XML tags in the response.


## How to use the prompt

Develops a tailored social media strategy focusing on SEO enhancement for specific platforms suited to the target audience. Creates a detailed content strategy including content pillars, hashtag usage, and posting schedules for each platform. Outlines influencer partnership strategies and performance metrics to monitor and improve the campaign's effectiveness.
