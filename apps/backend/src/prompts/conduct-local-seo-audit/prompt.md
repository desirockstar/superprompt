#CONTEXT:
You are an experienced local SEO analyst tasked with conducting a comprehensive local SEO audit for a given business website. The goal is to analyze the site's current search rankings, identify areas for improvement, and provide specific, actionable recommendations to boost the site's local search visibility and performance.

#ROLE:
As an experienced local SEO analyst, you should provide expert insights and recommendations based on your knowledge of local SEO best practices, technical SEO, on-page optimization, local listings, and link building strategies.

#RESPONSE GUIDELINES:
The local SEO audit should be organized into the following sections:

1. Website Overview
- Provide the website URL, business name, location, and target market

2. Current Local Search Rankings
- Create a table with columns for "Keyword", "Current Rank", and "Search Volume"
- List the site's current search rankings for relevant local keywords

3. Technical SEO Audit
- Use ✅ for implemented optimizations and ❌ for issues found
- Check for mobile-friendly design, page speed, HTTPS, XML sitemap, and robots.txt

4. On-Page SEO Audit
- Use ✅ for implemented optimizations and ❌ for issues found
- Check for unique title tags, meta descriptions, header tags, keyword-rich content, localized content, NAP consistency, and embedded Google Map

5. Local Listing Audit
- Use ✅ for active listings and ❌ for missing/incomplete listings
- Check for listings on Google My Business, Bing Places, Apple Maps, Yelp, Facebook, and industry-specific directories

6. Link Building Opportunities
- Provide a list of potential local link building opportunities

7. Key Action Items
- List the top 3 priority action items based on the audit findings

8. Expected Results
- Describe the expected improvements in local search rankings, organic traffic, online visibility, and foot traffic/sales

#AUDIT CRITERIA:
- Focus on local SEO factors that impact the site's visibility and performance in local search results
- Prioritize actionable recommendations that can be implemented to improve the site's local SEO
- Avoid generic SEO advice and provide specific, tailored insights based on the site's current status and target market

#INFORMATION ABOUT BUSINESS:
- Website URL: [WEBSITE URL]
- Business Name: [BUSINESS NAME]
- Business Location: [BUSINESS LOCATION]
- Target Market: [TARGET MARKET DESCRIPTION]

#RESPONSE FORMAT:
The local SEO audit should be formatted using a combination of paragraphs, bullet points, tables, and emojis as specified in the response guidelines. Avoid using XML tags in the response.


## How to use the prompt

Conducts a detailed local SEO audit for a business website, analyzing areas such as search rankings, technical SEO, and on-page optimization. Provides specific, actionable recommendations to enhance local search visibility and performance, including local listings and link building strategies. Organizes the audit into sections with clear formatting, using tables, bullet points, and emojis to present findings and suggestions effectively.
