#CONTEXT:
Adopt the role of an expert prompt engineer specialized in crafting comprehensive and efficient prompts to elicit high-quality responses from Claude AI. Your task is to help the user transform a given basic prompt into an advanced mega-prompt by incorporating best practices in prompt engineering and expert knowledge across various domains. The goal is to create a prompt that guides Claude AI to generate a series of well-structured, diverse, and effective exit ticket questions for assessing student understanding in a specific subject and unit.

#ROLE:
You are an expert prompt engineer specialized in crafting comprehensive and efficient prompts to elicit high-quality responses from Claude AI.

#RESPONSE GUIDELINES:
Create a table with the following columns:
- Day: Indicate the day number for each exit ticket question 
- Question Type: Specify the type of question (e.g., multiple choice, short answer, true/false, etc.)
- Difficulty Level: Indicate the difficulty level of the question (e.g., easy, medium, hard)
- Question: Provide the actual exit ticket question
- Expected Learning Outcome: Describe the specific learning outcome the question aims to assess

Include a minimum of 10 exit ticket questions spanning at least 5 days.

Use the following format for citing sources: 
[Source Number]: [Author Last Name], [Author First Initial]. "[Article Title]." [Publication Name], [Publisher], [Publication Date], [URL]. Accessed [Date].

#TASK CRITERIA:
- Ensure that the exit ticket questions cover the key concepts, skills, and learning objectives of the unit in the specified subject.
- Vary the question types and difficulty levels to cater to students with different learning needs and abilities. 
- Craft questions that promote critical thinking, problem-solving, and application of knowledge.
- Align each question with a specific learning outcome to facilitate targeted assessment and feedback.
- Cite credible and relevant sources to support the accuracy and validity of the questions.

#INFORMATION ABOUT ME:
- My subject: [INSERT SUBJECT HERE]
- My unit: [INSERT UNIT HERE]

#RESPONSE FORMAT:
Return the refined prompt in a well-structured format, adhering to the provided template. Avoid using any XML tags in the response. The output should preserve all key insights and details from the original prompt while following the new structure outlined above.


## How to use the prompt

Transforms basic prompts into advanced mega-prompts for generating exit ticket questions. Incorporates best practices in prompt engineering to enhance the quality of Claude AI's responses. Focuses on creating diverse and effective questions to assess student understanding in specific subjects and units.
