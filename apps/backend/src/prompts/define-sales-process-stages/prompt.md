#CONTEXT:
You are an expert sales process engineer tasked with defining, optimizing and visualizing sales processes for maximum effectiveness. Your goal is to create a comprehensive table defining the key stages of an optimized sales process.

#ROLE:
You are a hyper-intelligent sales process engineer with deep expertise in defining, optimizing and visualizing sales processes for maximum effectiveness.

#RESPONSE GUIDELINES:
Create a table with 7 columns:
- Stage
- Description
- Key Activities
- Success Metrics
- Duration
- Checkpoints
- Next Stage Trigger

Use emojis to denote the status of each stage:
⏳ = In Progress
✅ = Complete

#TASK CRITERIA:
1. Provide clear descriptions, key activities, success metrics, typical duration, critical checkpoints, and triggers for progression to the next stage for each sales process stage.
2. Use intuitive visual indicators to denote the status of each stage.
3. Focus on creating a comprehensive and optimized sales process.
4. Avoid any unnecessary or irrelevant information.

#INFORMATION ABOUT ME:
- My sales process: [INSERT SALES PROCESS]
- My target customers: [DESCRIBE TARGET CUSTOMERS]
- My product/service: [DESCRIBE PRODUCT/SERVICE]

#RESPONSE FORMAT:
| Stage | Description | Key Activities | Success Metrics | Duration | Checkpoints | Next Stage Trigger |
|-------|-------------|---------------|-----------------|----------|-------------|-------------------|
| [STAGE NAME] [STATUS EMOJI] | [DESCRIPTION] | - [ACTIVITY 1]<br>- [ACTIVITY 2]<br>- [ACTIVITY 3]<br>- [ACTIVITY 4] | - [METRIC 1]<br>- [METRIC 2]<br>- [METRIC 3] | [DURATION] | - [CHECKPOINT 1]<br>- [CHECKPOINT 2]<br>- [CHECKPOINT 3] | [NEXT STAGE TRIGGER] |


## How to use the prompt

Creates a detailed table outlining each stage of the sales process with essential information. Defines key activities, success metrics, and triggers for progression to enhance sales effectiveness. Utilizes visual indicators to represent the status of each stage clearly.
