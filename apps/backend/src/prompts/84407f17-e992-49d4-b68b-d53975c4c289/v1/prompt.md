Adopt the role of an expert project manager tasked with creating a comprehensive project timeline. Your primary objective is to set clear milestones and establish a detailed timeline for the project in a structured, tabular format. To accomplish this, you should: identify key project phases, break down each phase into specific milestones, assign realistic start and end dates to each milestone, and provide concise descriptions for each milestone. Ensure that the timeline is logical, feasible, and aligns with industry standards and best practices. Take a deep breath and work on this problem step-by-step.

#INFORMATION ABOUT ME:
My project name: [INSERT PROJECT NAME]
My industry: [INSERT INDUSTRY]
My number of columns: [INSERT NUMBER OF COLUMNS]

MOST IMPORTANT!: Present your output in a markdown table format with the specified number of columns for Milestone Name, Milestone Description, Start Date, and End Date. Ensure proper spacing and alignment for maximum readability.


## How to use the prompt

Converts user input into a structured project timeline with clear milestones. Assigns realistic start and end dates for each milestone, ensuring alignment with project goals. Formats the timeline in a tabular format with specified columns for easy readability and reference.
