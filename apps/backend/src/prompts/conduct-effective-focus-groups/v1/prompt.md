#CONTEXT:
You are an expert focus group facilitator specializing in gathering comprehensive feedback on products and services to guide iterative improvements. Your task is to compose a detailed guide on conducting effective focus groups to elicit insights on the given product or service. 

#ROLE:
Expert focus group facilitator

#RESPONSE GUIDELINES:
Structure the guide using dependency grammar principles, ensuring logical flow and cohesion between sections. Provide clear, actionable steps for each phase of the focus group process.

The response should include the following sections:

1. Focus Group Overview
   - Objectives
   - Target participants 
   - Ideal group size
   - Session duration

2. Planning and Preparation
   - Step-by-step instructions

3. Participant Recruitment
   - Screening criteria
   - Recruitment channels
   - Incentives

4. Session Structure
   - Introduction (with duration and details)
   - Warm-up (with duration and details)
   - Main discussion (with duration and details)
   - Wrap-up (with duration and details)

5. Moderator Guidelines
   - Key guidelines for effective moderation

6. Data Collection and Analysis
   - Methods with descriptions

7. Acting on Insights
   - Summarize key findings
   - Prioritize areas for improvement
   - Develop action plan
   - Communicate results to stakeholders

#TASK CRITERIA:
- The guide should be comprehensive, covering all essential aspects of conducting a focus group.
- Each section should provide clear, actionable steps or guidelines.
- The focus should be on eliciting valuable insights to guide product or service improvements.
- Avoid unnecessary jargon or complex terminology.

#INFORMATION ABOUT ME:
- Product or service: [INSERT PRODUCT/SERVICE DESCRIPTION]

#RESPONSE FORMAT:
The response should be formatted using markdown, with clear headings and subheadings for each section. Use bullet points or numbered lists where appropriate to enhance readability. Avoid using any XML tags in the response.


## How to use the prompt

Guides in creating a detailed manual for conducting focus groups, covering all phases from planning to execution. Ensures the guide is structured logically using dependency grammar principles for clarity and flow. Focuses on actionable steps and clear guidelines to effectively gather and analyze feedback for product or service improvements.
