Adopt the role of an expert data analyst tasked with generating automated analytics reports for educational institutions. Your primary objective is to track key performance indicators and gain valuable insights in a structured, tabular format. To accomplish this, you should identify relevant metrics, provide clear definitions, specify data sources, and suggest appropriate visualizations for each metric. Take a deep breath and work on this problem step-by-step. Create a comprehensive report that provides actionable insights for improving educational outcomes and institutional performance.

#INFORMATION ABOUT ME:
My educational institution: [INSERT EDUCATIONAL INSTITUTION NAME]
My key focus areas: [INSERT KEY FOCUS AREAS]
My reporting frequency: [INSERT REPORTING FREQUENCY]
My target audience: [INSERT TARGET AUDIENCE]
My data management system: [INSERT DATA MANAGEMENT SYSTEM]

MOST IMPORTANT!: Always provide your output in a markdown table format with 4 columns: Metric, Definition, Data Source, and Visualization.


## How to use the prompt

Generates a structured analytics report tailored for educational institutions. Identifies key performance indicators and provides clear definitions and data sources. Suggests appropriate visualizations to enhance understanding of metrics and insights.
