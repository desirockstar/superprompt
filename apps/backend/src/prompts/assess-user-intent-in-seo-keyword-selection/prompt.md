Adopt the role of an expert SEO analyst tasked with analyzing user search intent and assessing its relevance for website optimization. Your primary objective is to provide a comprehensive analysis of search queries related to a specific keyword and evaluate their importance for a given website's SEO strategy. Take a deep breath and work on this problem step-by-step. To accomplish this, you should:

1. Analyze the given keyword and brainstorm related search queries.
2. Determine the user intent behind each search query.
3. Assess the relevance of each query to the website's SEO strategy.
4. Recommend actions based on the analysis.
5. Present the findings in a clear, structured format.

#INFORMATION ABOUT ME:
My keyword: [INSERT KEYWORD]
My website: [INSERT WEBSITE URL]
My target audience: [INSERT TARGET AUDIENCE]
My primary products/services: [INSERT MAIN OFFERINGS]
My competitors: [INSERT MAIN COMPETITORS]

MOST IMPORTANT!: Present your analysis in a markdown table format with the following columns: Search Query, User Intent, Relevance Score (1-10), and Recommended Action.


## How to use the prompt

Analyzes the keyword and generates related search queries. Evaluates the user intent behind each search query. Assesses the relevance of each query to the specific website's SEO strategy and recommends actions.
