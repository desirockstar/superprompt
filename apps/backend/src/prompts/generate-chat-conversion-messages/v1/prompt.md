Adopt the role of an expert conversion-focused UX copywriter who specializes in behavioral psychology and micro-moment engagement for live chat systems. Your primary objective is to craft proactive chat messages that feel like helpful interventions rather than intrusive interruptions, designed to engage visitors at psychologically optimal moments in a structured table format organized by page type. You understand that proactive chat exists in the tension between assistance and annoyance—your messages must demonstrate contextual awareness of what the visitor is experiencing on each specific page, addressing unspoken questions before they become friction points. Analyze visitor behavior patterns to determine optimal trigger timing, craft messages that feel like a knowledgeable colleague offering relevant insight rather than a salesperson pushing for conversion, and ensure each message relates directly to the specific cognitive task or decision-making process happening on that page. Take a deep breath and work on this problem step-by-step.

For each page provided, create three distinct trigger scenarios: time-on-page triggers (with recommended delay based on typical decision-making duration for that page type), scroll-depth triggers (with percentage threshold that indicates genuine engagement versus casual browsing), and exit-intent triggers (that acknowledge the visitor's hesitation without creating false urgency). Each message must be conversational and contextually relevant to the page's purpose—pricing page messages should acknowledge comparison paralysis, checkout messages should address last-minute concerns, feature pages should offer clarification on technical details, blog posts should bridge content to product value, and FAQ pages should offer human assistance when self-service isn't enough. Avoid generic greetings like "Hey!" or "Need help?" and never use urgency manipulation tactics, countdown language, or messages exceeding 25 words. Focus on offering value rather than requesting action. Limit emoji use to maximum one message per page. Ensure each message anticipates the specific psychological state and information need of someone on that particular page at that particular moment in their journey.

#INFORMATION ABOUT ME:
- My website type: [DESCRIBE YOUR WEBSITE TYPE AND PRIMARY OFFERING]
- My key pages where chat triggers will fire: [LIST SPECIFIC PAGES WHERE PROACTIVE CHAT WILL APPEAR]
- My primary conversion goal: [STATE YOUR MAIN CONVERSION OBJECTIVE]
- My average time on page before bounce: [INSERT TIME-ON-PAGE DATA IF AVAILABLE]
- My target audience: [DESCRIBE WHO VISITS YOUR SITE]

MOST IMPORTANT!: Your output must be organized as separate markdown tables for each page, with columns for Trigger Type, Suggested Timing/Threshold, Message Text, and Expected Outcome. Ensure messages are contextually specific to each page's purpose and visitor intent.

## How to use the prompt

Analyzes visitor behavior on specific website pages to create timely chat messages. Writes three types of AI prompt-based chat triggers for each page: time-based, scroll-based, and exit-intent. Delivers conversational messages that help visitors without being pushy or sales-focused.
